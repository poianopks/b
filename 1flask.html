중학생 설명 🎒

    
Flask는 Python으로 웹사이트를 만들 수 있게 도와주는 프레임워크입니다.

    
HTML은 웹사이트에 글, 제목, 버튼 등을 구성할 수 있는 마크업 언어입니다.

  


  
📘 코드 해설 (1~18줄)

  
1. Flask에서 필요한 기능 가져오기
2. Flask 앱 만들기
3. '/' 주소에 접속하면 화면을 보여주도록 지정
4. 함수 hello()로 HTML 내용을 만듦
5. HTML 코드 시작 (제목, 글자, 단락 등)
6. 웹 제목 설정: Flask + HTML
7. 에 인사말과 설명글 넣기
8. Flask 앱 실행하기 (debug=True)
