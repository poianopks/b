1. HTML이란 무엇일까?

HTML은 웹페이지를 만드는 언어예요. 마치 레고 블록처럼, 여러 가지 조각을 쌓아서 멋진 페이지를 만들 수 있어요!

따라하기

HTML은 "HyperText Markup Language"의 약자예요. 컴퓨터가 이해할 수 있는 말로, 웹페이지를 만드는 데 사용해요. 예를 들어, 제목과 문단을 만들 수 있어요.

<h1>안녕!</h1>
<p>이건 문단이야.</p>

위 코드를 쓰면 "안녕!"이라는 큰 제목과 "이건 문단이야."라는 문장이 나타나요!

2. 첫 웹페이지 만들기

첫 번째 웹페이지를 만들어보아요. 간단한 코드로 "안녕하세요!"라고 써보는 거예요.

따라하기

웹페이지를 만들려면 기본 구조가 필요해요. 아래 코드를 따라 써보세요:

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>내 첫 페이지</title>
</head>
<body>
    <h1>안녕하세요!</h1>
</body>
</html>

이 코드를 파일로 저장해서 열어보면 "안녕하세요!"라는 제목이 나타나요!

3. 텍스트 추가하기

웹페이지에 더 많은 텍스트를 추가해보아요. 제목과 문단을 만드는 방법을 배워봐요.

따라하기

제목은 <h1> 태그, 문단은 <p> 태그로 만들어요. 아래를 따라 해보세요:

<h1>내가 좋아하는 것</h1>
<p>나는 사과를 좋아해요.</p>
<p>바나나도 좋아해요!</p>

이렇게 하면 큰 제목과 두 개의 문단이 생겨요!

4. 글씨 크기 바꾸기

글씨를 더 크게 또는 작게 만들 수 있어요. 제목은 크게, 설명은 작게 해보아요.

따라하기

CSS라는 도구로 글씨 크기를 바꿀 수 있어요. 아래 코드를 써보세요:

<style>
    h1 { font-size: 40px; }
    p { font-size: 20px; }
</style>
<h1>큰 제목</h1>
<p>작은 문단</p>

이 코드는 제목을 40px, 문단을 20px 크기로 만들어줘요!

5. 글씨 색깔 바꾸기

글씨에 색깔을 입혀보아요. 빨간색, 파란색, 노란색 등 좋아하는 색을 골라봐요.

따라하기

CSS로 글씨 색깔을 바꿀 수 있어요. 아래를 따라 해보세요:

<style>
    h1 { color: red; }
    p { color: blue; }
</style>
<h1>빨간 제목</h1>
<p>파란 문단</p>

이렇게 하면 제목은 빨간색, 문단은 파란색으로 변해요!

6. 그림 넣기

웹페이지에 그림을 넣는 방법을 배워봐요. 좋아하는 사진을 올려보세요!

따라하기

<img> 태그로 그림을 넣어요. 아래 코드를 써보세요:

<img src="cat.jpg" alt="고양이 사진">

"cat.jpg" 대신 좋아하는 그림 파일 이름을 넣으면 그림이 나타나요!

7. 목록 만들기

목록을 만들어보아요. 좋아하는 음식이나 장난감을 나열해봐요.

따라하기

<ul>와 <li> 태그로 목록을 만들어요. 아래를 따라 해보세요:

<ul>
    <li>피자</li>
    <li>아이스크림</li>
    <li>케이크</li>
</ul>

이렇게 하면 맛있는 음식 목록이 생겨
