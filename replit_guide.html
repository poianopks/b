Replit - 앱 사용법 따라하기

시작하기 HTML 코딩 Python 코딩 코드 실행 공유 방법

Replit 시작하기

1. Replit 앱 또는 웹사이트(replit.com)를 엽니다.

2. 구글 계정 등으로 로그인합니다.

3. Create Repl을 눌러 새 프로젝트를 시작합니다.

HTML 코딩

1. 언어에서 HTML, CSS, JS 템플릿을 선택해요.

2. index.html 파일에 HTML 코드를 입력합니다.

<h1>안녕하세요!</h1>
<p>Replit에서 만든 첫 웹페이지예요.</p>

Python 코딩

1. Create Repl → Python을 선택합니다.

2. main.py에 아래처럼 코딩해요.

print("Hello, Replit!")

코드 실행

코드를 다 쓴 후, 상단의 Run 버튼을 누르면 코드가 실행돼요!

공유 방법

Replit 프로젝트를 친구와 공유하려면:

  1. 우측 상단의 Share 버튼 클릭
  2. 링크 복사해서 친구에게 보내기
