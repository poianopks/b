서비스 1 · 상세 학습 설명

개념부터 원리까지, 처음 배우는 분을 위한 완전 해설

🎯 이 영상은 무엇을 다루나요?

이 영상은 IT 전문가이자 베스트셀러 작가인 최원영과 함께 챗GPT(ChatGPT)와 파이썬(Python)을 이용하여 주식 투자에 필요한 정보를 자동으로 수집하고 엑셀 파일로 정리하는 방법을 시연합니다.

핵심은 금융 정보 사이트인 인베스팅닷컴(investing.com)에서 약 1,000개~2,500개 종목의 기술적 분석(매수·매도·중립) 데이터를 웹 크롤링으로 자동 수집하여, 날짜와 함께 엑셀 파일에 저장하는 전 과정을 보여주는 것입니다.

영상에서는 코드 파일과 2,500개 종목의 분석 결과 엑셀 파일을 무료로 제공하며, 비전공자도 이해할 수 있도록 쉽게 설명합니다.

🧩 핵심 개념 4가지

개념한 줄 정의이 영상에서의 역할
웹 크롤링
(Web Crawling)		 자동화 프로그램이 웹사이트를 돌아다니며 정보를 수집하는 기술 인베스팅닷컴에서 종목별 기술적 분석 데이터를 자동으로 긁어옴
파이썬
(Python)		 배우기 쉽고 강력한 프로그래밍 언어 크롤링 코드 작성 + 데이터 정리 + 엑셀 저장 전 과정을 담당
챗GPT
(ChatGPT)		 코드 작성, 디버깅, 아이디어 제안을 도와주는 AI 비서 파이썬 크롤링 코드를 대신 작성해주는 역할
기술적 분석
(Technical Analysis)		 과거 가격·거래량 데이터로 미래 주가를 예측하는 방법 인베스팅닷컴의 '적극 매수/매도' 의견이 수집 대상 데이터

⚙️ 자동화 프로세스 5단계

1

목표 설정

분석할 주식 종목 목록을 정합니다. 영상에서는 코스피·코스닥 전체 종목(약 2,500개)을 대상으로 합니다.

2

데이터 소스 확인

인베스팅닷컴의 기술적 분석 페이지 구조를 파악합니다. 브라우저의 '개발자 도구'로 원하는 정보가 어떤 HTML 태그에 담겨 있는지 확인합니다.

3

챗GPT로 코드 작성

챗GPT에게 목표를 설명하고 파이썬 코드를 생성받습니다. 처음에는 단일 종목, 이후 전체 종목으로 점진적으로 발전시킵니다.

4

크롤링 실행

작성된 코드를 실행하면 프로그램이 각 종목 페이지에 순차적으로 접속하여 기술적 분석 의견을 자동으로 수집합니다.

5

엑셀로 저장 및 분석

수집된 데이터(종목명, 날짜, 기술적 분석 의견)를 엑셀 파일로 저장합니다. 이를 통해 한눈에 시장 전체를 조망하고 투자 대상을 선별할 수 있습니다.

🚀 이 시스템의 기대 효과

이 자동화 시스템을 구축하면 수천 개 종목의 정보를 수동으로 확인하는 데 드는 막대한 시간을 절약할 수 있습니다. 모든 종목을 동일한 데이터 기준으로 평가하므로 특정 종목에 대한 감정적 편향을 배제하고 객관적인 투자 판단을 내릴 수 있습니다.

또한 매일 데이터를 축적하면 전반적인 시장 흐름이나 특정 섹터의 동향을 파악하는 데도 활용할 수 있습니다. 이는 단순한 정보 수집을 넘어, 데이터 기반 투자 습관을 형성하는 첫걸음이 됩니다.

서비스 2 · 왕초보 따라하기 가이드

이 가이드를 시작하기 전에: 코딩 경험이 전혀 없어도 괜찮습니다. 정해진 순서대로 블록을 조립하듯 따라 하기만 하면 됩니다.

🔧 1단계 · 필수 프로그램 설치

1

파이썬(Python) 설치

python.org/downloads에 접속하여 최신 버전을 다운로드합니다.

⚠️ 중요: 설치 화면에서 "Add Python to PATH" 체크박스를 반드시 체크하고 설치하세요. 이 옵션이 없으면 파이썬이 제대로 동작하지 않습니다.
2

VS Code 설치

code.visualstudio.com에서 Visual Studio Code를 다운로드하여 설치합니다. 코드를 작성하고 실행하는 작업 공간입니다. 완전 무료입니다.

3

파이썬 라이브러리 설치

VS Code를 열고 상단 메뉴 터미널(Terminal) → 새 터미널(New Terminal)을 클릭합니다. 화면 하단에 나타난 터미널 창에 아래 명령어를 한 줄씩 입력하고 엔터를 누릅니다.

🤖 2단계 · 챗GPT에게 코드 부탁하기

1

챗GPT 접속 및 로그인

chat.openai.com에 접속하여 회원가입 및 로그인합니다. 기본 무료 버전으로 충분합니다.

2

명확하게 요청하기

챗GPT 대화창에 아래 내용을 그대로 복사하여 붙여넣고 전송합니다.

investing.com 웹사이트에서 삼성전자 주식의 기술적 분석 요약 정보(예: '적극 매수', '매도')를 가져오는 파이썬 코드를 만들어줘. requests와 beautifulsoup4 라이브러리를 사용하고, 결과를 엑셀 파일로 저장해줘.
3

코드 복사하기

챗GPT가 코드를 생성해주면, 코드 블록 우측 상단의 "Copy" 버튼을 클릭하여 복사합니다. 처음에는 코드 내용을 완전히 이해하지 못해도 괜찮습니다.

🔍 3단계 · 정보 위치 찾기 (개발자 도구 활용)

웹 크롤링은 웹사이트에서 원하는 정보의 정확한 위치를 파악해야 합니다. 아래 순서를 따라 해보세요.

1

인베스팅닷컴 접속

크롬 브라우저에서 kr.investing.com에 접속하여 분석하고 싶은 주식(예: 삼성전자)을 검색합니다.

2

'기술 분석' 탭 클릭

해당 종목 페이지에서 상단 탭 중 '기술 분석'을 클릭합니다. '요약' 항목에 '적극 매수', '매도' 등의 의견이 표시됩니다.

3

개발자 도구로 위치 확인

수집하고 싶은 텍스트(예: '적극 매수') 위에서 마우스 오른쪽 버튼 클릭 → '검사'를 선택합니다. 오른쪽에 나타난 HTML 코드에서 해당 텍스트를 감싸고 있는 태그와 class 이름을 확인합니다. 이 정보를 챗GPT에게 알려주면 더 정확한 코드를 받을 수 있습니다.

💻 4단계 · 코드 붙여넣기 및 실행

1

새 파일 만들기

VS Code에서 Ctrl + N (맥: Cmd + N)을 눌러 새 파일을 만들고, stock_crawler.py라는 이름으로 저장합니다.

2

코드 붙여넣기

아래는 핵심 로직을 담은 예시 코드입니다. 챗GPT가 생성한 코드 또는 아래 예시를 파일에 붙여넣으세요.

import requests from bs4 import BeautifulSoup import pandas as pd from datetime import datetime # 분석할 종목 목록 (종목명: 인베스팅닷컴 URL 코드) STOCK_CODES = { '삼성전자': 'samsung-electronics-co-ltd', 'SK하이닉스': 'sk-hynix-inc', } results = [] headers = {'User-Agent': 'Mozilla/5.0'} for name, code in STOCK_CODES.items(): try: url = f'https://kr.investing.com/equities/{code}-technical' response = requests.get(url, headers=headers) soup = BeautifulSoup(response.text, 'html.parser') # 기술적 분석 요약 값 추출 summary_el = soup.find('span', {'data-test': 'summary-value'}) summary = summary_el.text.strip() if summary_el else '정보 없음' today = datetime.now().strftime('%Y-%m-%d') results.append([name, today, summary]) print(f'{name}: {summary}') except Exception as e: print(f'{name} 오류: {e}') # 엑셀 파일로 저장 df = pd.DataFrame(results, columns=['종목명', '날짜', '기술적 분석']) df.to_excel('stock_results.xlsx', index=False) print('완료! stock_results.xlsx 파일을 확인하세요.')
3

실행하기

VS Code 터미널에서 아래 명령어를 입력하고 엔터를 누릅니다.

python stock_crawler.py

터미널에 종목별 분석 결과가 출력되면 성공입니다!

코드가 모두 실행되면 stock_crawler.py 파일이 있는 폴더에 stock_results.xlsx 엑셀 파일이 생성됩니다. 파일을 열면 아래와 같은 형태로 데이터가 정리되어 있습니다.

서비스 3 · 핵심 팁 요약

6가지 핵심 질문으로 이 시스템을 완전히 이해하세요

Q1 이게 정확히 무엇인가요?
챗GPT와 파이썬을 이용해 수많은 주식 종목의 투자 정보를 자동으로 수집하고 엑셀 파일로 정리해주는 개인용 자동화 시스템입니다. 금융 정보 사이트 인베스팅닷컴의 기술적 분석 데이터를 예시로 사용하며, 한 번 만들어두면 버튼 클릭 한 번으로 수천 개 종목의 최신 '매수/매도' 의견을 순식간에 취합할 수 있습니다.
Q2 왜 필요한가요?
성공적인 투자를 위해서는 개인의 감이나 근거 없는 소문이 아닌, 객관적인 데이터에 기반한 판단이 필수적이기 때문입니다. 하지만 수천 개 종목의 데이터를 매일 사람이 직접 확인하는 것은 현실적으로 불가능합니다. 이 자동화 시스템은 그 불가능을 가능하게 만들어, 투자자가 더 넓은 시야로 시장을 보고 합리적인 의사결정을 내릴 수 있도록 돕습니다.
Q3 어떤 도움이 되나요?
혜택설명
압도적인 시간 절약수천 개 종목 정보 수집을 단 몇 분 만에 완료
객관성 확보모든 종목을 동일한 기준으로 평가, 감정적 편향 제거
새로운 기회 발견평소 관심 없던 종목에서도 좋은 신호 포착 가능
투자 습관 개선'감' 투자에서 데이터 기반 투자 루틴으로 전환
Q4 작동하고 실행하기 어렵지 않나요?
왕초보도 충분히 따라 할 수 있습니다. 영상의 핵심 메시지는 전문 개발자가 아니더라도 챗GPT와 같은 인공지능의 도움을 받으면 누구나 자신에게 필요한 프로그램을 만들 수 있다는 것입니다. 어려운 수학 문제를 푸는 것이 아니라, 정해진 순서대로 블록을 조립하는 것에 가깝습니다. 위의 '왕초보 따라하기 가이드(서비스 2)'를 그대로 따라 하면 됩니다.
Q5 돈이 많이 드나요?
아니요, 영상에서 사용된 모든 도구는 완전 무료입니다.
도구비용비고

인베스팅닷컴무료기본 기술적 분석 데이터 무료 제공

Q6 어디서 실행하는 건가요?
이 시스템은 사용자 개인의 컴퓨터(PC 또는 노트북)에서 직접 실행됩니다. 클라우드 서버나 별도의 호스팅 서비스가 필요 없습니다. '왕초보 따라하기 가이드'에 따라 파이썬과 VS Code를 설치한 여러분의 컴퓨터가 바로 이 강력한 분석 도구의 실행 환경이 됩니다. 인터넷만 연결되어 있다면 언제 어디서든 실행할 수 있습니다.
