4. 왕초보 30가지 (클릭하면 설명)

1. 파이썬이란?
1991년 귀도 반 로섬이 만든 쉽고 강력한 프로그래밍 언어. 영어처럼 읽히는 문법으로 왕초보도 1주일 만에 실무 가능.
2. 설치 방법 (윈도우)
1) python.org 접속 2) Python 3.12 다운 → 설치 시 “Add to PATH” 체크 3) 터미널에 python 입력 → 버전 나오면 성공!
3. 첫 프로그램 “Hello, World!”
print("Hello, World!")
저장 → python hello.py 실행!
4. 변수와 타입
name = "삼성"; price = 70600; per = 10.8
5. 숫자 연산
a = 10; b = 3; print(a + b, a ** b)
6. 문자열 다루기
msg = "삼성전자"; print(msg[0], msg[-2:])
7. 리스트 기초
prices = [70000, 70500]; prices.append(71000)
8. 딕셔너리 기초
samsung = {"code":"005930","price":70600}
9. 조건문 if
if price > 70000: print("상승")
10. 반복문 for
for p in [70000, 70500]: print(p)
11. while 문
i = 1; while i <= 3: print(i); i += 1
12. 함수 만들기
def add_tax(price): return price * 1.1
13. 패키지 설치 pip
터미널에서 pip install yfinance pandas matplotlib
14. 주식 데이터 가져오기 (yfinance)
import yfinance as yf; df = yf.download('005930.KS'); print(df.tail(1))
15. 차트 그리기
import matplotlib.pyplot as plt; plt.plot(df['Close']); plt.show()
16. 뉴스 제목 가져오기
from newspaper import Article; a=Article(url,language='ko'); a.download(); a.parse(); print(a.title)
17. CSV 파일 읽기/쓰기
import pandas as pd; df=pd.read_csv('price.csv'); df.to_csv('result.csv')
18. 에러 처리 try/except
try: int("삼") except ValueError: print("숫자만 입력하세요")
19. 날짜 다루기 (datetime)
from datetime import date; print(date.today().strftime("%Y-%m-%d"))
20. 리스트 컴프리헨션
squares=[x**2 for x in range(5)]
21. 람다 함수
multiply=lambda x,y:x*y
22. enumerate, zip
for idx,val in enumerate(['A','B']): print(idx,val)
23. set 중복 제거
nums=[1,2,2,3]; print(set(nums))
24. JSON 읽기/쓰기
import json; json.dump({"price":70600},open('s.json','w')); d=json.load(open('s.json'))
