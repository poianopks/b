파이썬 왕초보를 위한 기초 공부

30개 주제로 배우는 파이썬 프로그래밍

각 주제를 클릭하면 상세 설명과 예제 코드를 볼 수 있습니다. 직접 코드를 작성해보며 파이썬 기초를 탄탄히 다져보세요!

1. 변수(Variables)
1
변수는 데이터를 저장하는 공간입니다. 파이썬에서는 변수를 선언할 때 데이터 타입을 명시하지 않습니다.
# 변수 선언 및 사용 예제
name = "김파이" # 문자열 변수
age = 10 # 정수형 변수
height = 150.5 # 실수형 변수
is_student = True # 불리언 변수

print("이름:", name)
print("나이:", age)
print("키:", height)
print("학생인가요?", is_student)
2. 데이터 타입(Data Types)
2
파이썬의 기본 데이터 타입에는 정수(int), 실수(float), 문자열(str), 불리언(bool) 등이 있습니다.
# 다양한 데이터 타입 예제
num = 10 # int (정수)
pi = 3.14 # float (실수)
name = "Python" # str (문자열)
is_cool = True # bool (불리언)

# 타입 확인하기
print(type(num)) # <class 'int'>
print(type(pi)) # <class 'float'>
print(type(name)) # <class 'str'>
print(type(is_cool)) # <class 'bool'>
3. 연산자(Operators)
3
파이썬에는 산술, 비교, 논리 연산자 등이 있습니다.
# 산술 연산자
a = 10
b = 3
print("a + b =", a + b) # 13
print("a - b =", a - b) # 7
print("a * b =", a * b) # 30
print("a / b =", a / b) # 3.333...
print("a % b =", a % b) # 1 (나머지)
print("a ** b =", a ** b) # 1000 (제곱)

# 비교 연산자
print("a == b?", a == b) # False
print("a != b?", a != b) # True
print("a > b?", a > b) # True

# 논리 연산자
x = True
y = False
print("x and y:", x and y) # False
print("x or y:", x or y) # True
print("not x:", not x) # False
4. 조건문(if문)
4
조건문은 특정 조건에 따라 다른 코드를 실행할 수 있게 해줍니다.
age = 18

if age >= 19:
    print("성인입니다. 주류를 구매할 수 있습니다.")
elif age >= 13:
    print("청소년입니다.")
else:
    print("어린이입니다.")

# 중첩 if문 예제
score = 85
if score >= 90:
    print("A학점")
    if score >= 95:
        print("참 잘했어요!")
elif score >= 80:
    print("B학점")
elif score >= 70:
    print("C학점")
else:
    print("F학점")
5. 반복문(for문)
5
for문은 컬렉션(리스트, 튜플, 문자열 등)의 요소들을 순회하며 반복 작업을 수행합니다.
# 리스트 순회
fruits = ["사과", "바나나", "체리"]
for fruit in fruits:
    print(fruit)

# range() 함수 사용
# 0부터 4까지 출력
for i in range(5):
    print(i)

# 1부터 10까지 짝수 출력
for num in range(2, 11, 2):
    print(num)

# 중첩 for문 (구구단)
for i in range(2, 10):
    print(f"--- {i}단 ---")
    for j in range(1, 10):
        print(f"{i} x {j} = {i*j}")
6. 반복문(while문)
6
while문은 조건이 참인 동안 코드 블록을 반복 실행합니다.
# 기본 while문
count = 0
while count < 5:
    print(count)
    count += 1 # count = count + 1

# 무한루프와 break
while True:
    user_input = input("종료하려면 'exit' 입력: ")
    if user_input == "exit":
        print("프로그램을 종료합니다.")
        break

# continue 사용 예제
num = 0
while num < 10:
    num += 1
    if num % 2 == 0: # 짝수일 때
        continue # 다음 반복으로 건너뜀
    print(num) # 홀수만 출력
7. 리스트(Lists)
7
리스트는 순서가 있는 여러 항목을 저장하는 컬렉션입니다. 변경 가능한(mutable) 데이터 타입입니다.
# 리스트 생성
colors = ["빨강", "주황", "노랑", "초록", "파랑"]
numbers = [1, 3, 5, 7, 9]
mixed = [10, "텍스트", True, 3.14]

# 인덱싱
print(colors[0]) # "빨강"
print(colors[-1]) # "파랑" (마지막 요소)

# 슬라이싱
print(colors[1:3]) # ["주황", "노랑"]
print(colors[:2]) # ["빨강", "주황"]
print(colors[3:]) # ["초록", "파랑"]

# 리스트 수정
colors[2] = "금색" # 인덱스 2의 값 변경
colors.append("보라") # 끝에 추가
colors.insert(1, "분홍") # 인덱스 1에 삽입
colors.remove("주황") # 항목 제거

# 리스트 길이
print(len(colors)) # 6
8. 튜플(Tuples)
8
튜플은 리스트와 비슷하지만 변경 불가능한(immutable) 데이터 타입입니다. 한 번 생성된 튜플은 수정할 수 없습니다.
# 튜플 생성
rgb = ("빨강", "초록", "파랑")
coordinates = (37.5665, 126.9780) # 서울 좌표

# 인덱싱과 슬라이싱
print(rgb[0]) # "빨강"
print(rgb[1:3]) # ("초록", "파랑")

# 튜플은 수정 불가능
# rgb[0] = "분홍" # 에러 발생!

# 튜플 언패킹
r, g, b = rgb
print(r) # "빨강"
print(g) # "초록"
print(b) # "파랑"

# 튜플을 사용하는 경우
# - 변경되지 않아야 하는 데이터
# - 함수에서 여러 값 반환
def get_user():
    return "홍길동", 25, "서울"

name, age, city = get_user()
print(f"이름: {name}, 나이: {age}, 지역: {city}")
9. 딕셔너리(Dictionaries)
9
딕셔너리는 키-값 쌍으로 데이터를 저장하는 컬렉션입니다. 순서가 없으며 키를 통해 값을 빠르게 찾을 수 있습니다.
# 딕셔너리 생성
student = {
    "이름": "김파이",
    "학번": "20230001",
    "전공": "컴퓨터공학",
    "학년": 2
}

# 값 접근
print(student["이름"]) # "김파이"
print(student.get("학번")) # "20230001"

# 키 존재 여부 확인
if "전공" in student:
    print("전공 정보 있음")

# 항목 추가/수정
student["학점"] = 3.8 # 추가
student["학년"] = 3 # 수정

# 항목 삭제
del student["학번"]

# 모든 키와 값 순회
for key in student:
    print(key, ":", student[key])

# 키-값 쌍으로 순회
for key, value in student.items():
    print(key, ":", value)
10. 집합(Sets)
10
집합은 중복을 허용하지 않고 순서가 없는 컬렉션입니다. 교집합, 합집합 등의 수학적 연산을 지원합니다.
# 집합 생성
fruits = {"사과", "바나나", "오렌지"}
numbers = set([1, 2, 3, 4, 5])

# 중복은 자동으로 제거됨
colors = {"빨강", "파랑", "빨강", "초록"}
print(colors) # {'빨강', '파랑', '초록'}

# 항목 추가/삭제
fruits.add("포도")
fruits.remove("바나나")

# 집합 연산
A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}

print(A | B) # 합집합: {1, 2, 3, 4, 5, 6, 7, 8}
print(A & B) # 교집합: {4, 5}
print(A - B) # 차집합: {1, 2, 3}
print(A ^ B) # 대칭차집합: {1, 2, 3, 6, 7, 8}

# 부분집합 확인
C = {1, 2, 3}
print(C.issubset(A)) # True
