🎯 섹션 1: 김 AI 버전 업그레이드 (1.5 → 2025년 현재)

중요! "K2"라고 적으신 것은 "Kimi 2"라는 뜻으로 보입니다. 김 AI는 2024년부터 큰 변화가 있었어요.
기능 항목 1.5 버전 (예전) 2025년 현재 버전 (새로워진 점)
대화 능력 일반적인 대화만 가능 '김리서처(Kimi-Researcher)' 기능 추가 - 스스로 인터넷 검색하고 분석해줌
파일 업로드 파일 10개까지만 업로드 최대 50개 파일 동시 업로드 가능
답변 형식 텍스트 위주 답변 표, 그래프, 구조화된 답변 만들어줌
정보 제공 수준 단순 정보 제공 심층 분석 + 해결책 제시 + 실행 계획까지 한 번에
문서 처리 한 번에 20만자 정도 읽기 더 긴 문서도 빠르게 이해하고 요약
쉽게 말하면: 옛날에는 "선생님, 이거 알려주세요" 하면 "네, 이래요"라고만 답했는데, 이제는 "선생님, 이거 알려주세요" 하면 "네, 이래요. 그리고 이렇게 쓰면 되고, 여기에 더 궁금한 점이 있으면 이거 보세요, 마지막으로 이렇게 실행하면 돼요"라고 옆에서 끝까지 도와주는 스마트한 조교가 된 거예요!

🔧 섹션 2: 김 AI의 주요 기능 (코딩/주식/URL/유튜브/파일)

코딩 도움 (프로그래밍)

예시 상황: "초보자인데 파이썬으로 주식 차트 그리고 싶어요!"
  1. 코드 짜기: "아래 코드를 복사해서 붙여넣으세요"라며 완전한 코드를 만들어줌
  2. 설명하기: "이 줄은 뭐고, 저 줄은 뭐야"라고 한 줄씩 쉽게 설명
  3. 에러 고치기: 오류 메시지 복사해서 붙이면 "아, 여기 세미콜론(;) 빠졌어요!"라며 해결법 알려줌
  4. 개선하기: "이 코드를 더 빠르게 만들려면 이렇게 하세요"라며 최적화 방법 제시
# 실제 김 AI가 만들어주는 코드 예시 import matplotlib.pyplot as plt import pandas as pd # 주식 데이터 만들기 (예시) data = { '날짜': ['2025-11-01', '2025-11-02', '2025-11-03'], '가격': [50000, 51000, 49500] } df = pd.DataFrame(data) df.plot(x='날짜', y='가격', title='주식 차트') plt.show()

주식 분석 도움

예시 상황: "삼성전자 주식이 이제 오를까요?"
  1. 실시간 정보 수집: 김 AI가 인터넷에서 최신 뉴스, 재무제표, 공시를 찾아봄
  2. 다각도 분석: "삼성전자의 최근 실적은 이래요, 반도체 산업 전망은 이래요, 경쟁사는 이래요"라며 10가지 관점에서 분석
  3. 리스크 경고: "하지만 이런 위험도 있으니 조심하세요"라고 반대 의견도 함께 알려줌
  4. 정리: "결론: 단기는 불확실하지만, 장기는 긍정적. 투자 비중은 전체의 10% 정도가 적당"처럼 명확한 요약 제공
중요한 주의사항: 김 AI는 투자 조언을 해주지 않습니다! 정확히는 "이런 정보가 있어요, 이렇게 분석할 수 있어요"라고 도구와 정보를 제공할 뿐, "사라" "팔아라"는 말 안 해요. 투자는 본인 판단이 필요해요!

사이트 URL 분석

사용법:
1. 김 AI 창에 "https://~~~~" 주소를 그대로 붙여넣기
2. 뒤에 "이 사이트 뭐하는 곳이야? 요약해줘" 라고 추가
3. 전송!
# 사용 예시 https://www.naver.com 이 사이트는 뭐하는 곳이고, 주요 서비스 5개만 알려줘

유튜브 주소 요약

사용법:
  1. 유튜브에서 "공유" 버튼 누르고 "링크 복사"
  2. 김 AI 창에 붙여넣기 + "이 영상 3줄로 요약해줘"
  3. 전송!
유용한 팁:
- "타임스탬프 만들어줘" 라고 하면 "3분 10초: 이런 내용, 7분 25초: 이런 내용"처럼 목차 만들어 줌
- "영상에서 말한 방법으로 실습해볼게. 가이드 만들어줘" 라고 하면 실행 가능한 가이드 제작

첨부파일 분석

지원 파일: PDF, Word, TXT, PPT, Excel, 이미지(JPG, PNG, GIF), 코드(Python, Java, C++, HTML), ZIP (안에 위 파일들이 있으면)
파일 타입 분석 방법 예시 질문
PDF 논문 파일 올리고 질문 "이 논문 10페이지인데, 연구 방법 부분만 쉽게 설명해줘. 고등학생도 이해할 수 있게"
엑셀 주식 데이터 드래그 앤 드롭 "이 엑셀 파일에 있는 삼성전자 주가 데이터를 보고, 3가지 투자 관찰점을 알려줘"
이미지 사진 찍어서 올리기 "이 사진에 있는 차트 3개를 읽어서 표로 만들어줘"
코드 파일 .py 파일 직접 업로드 "이 파이썬 코드가 뭘 하는지 모르겠어. 5줄 요약 + 문제점 있으면 알려줘"
김 AI의 강점: 최대 50개 파일 동시 분석, 각 파일 100MB까지 가능, 200만자까지 읽을 수 있음 (책 20권 분량), 파일 간 비교도 가능 ("이 두 보고서 차이점 알려줘")

⚔️ 섹션 3: 챗GPT, 마누스 AI와 차별점 및 장단점

비교 항목 김 AI (Kimi) 챗GPT (ChatGPT) 마누스 AI (Manus)
출신 중국 月之暗面(Moonshot AI) 미국 OpenAI 중국 Manus AI
최대 강점 超长 문서 처리 (200만자) 코딩 능력 매우 강함 Agent 자율 실행 (다른 사이트에서 작업)
한국어 매우 자연스러움 자연스러움 양호
파일 분석 최대 50개, 100MB 제한적 (GPT-4는 32MB) 파일 기능 제한적
인터넷 검색 실시간 검색 매우 강함 일부 버전만 가능 자율 검색 가능
유튜브 요약 자막 기반 정밀 요약 링크만 인식 제한적
코딩 품질 매우 좋음 최고 수준 좋음
주식 분석 한국 주식에 최적화 일반적 일반적
사용 방법 웹, 앱, API 웹, 앱, API 초대장 필요
무료/유료 대부분 무료 일부 무료, 대부분 유료 현재 무료

김 AI 장점 🌟

  1. 📄 超长 문서: 200매 보고서도 한 번에 읽어줘요 (챗GPT는 자르고 여러 번 물어봐야 함)
  2. 🇰🇷 한국어 강점: 한국 뉴스, 한국 주식, 한국 문화를 더 잘 이해해요
  3. 📂 파일 많이: 회사 보고서 50개를 한 번에 올려서 "이 중에서만 요약해줘" 할 수 있어요
  4. 🆓 무료가 강력: 대부분 기능이 무료지만 성능은 유료 AI 못지않음
  5. 🔍 검색 빠름: 최신 정보도 10초 안에 찾아서 정리해줘요

김 AI 단점 ⚠️

  1. 💻 코딩은 챗GPT가 약간 더 낫: 특히 복잡한 알고리즘 문제
  2. 🧮 수학 계산 가끔 틀림: 계산기보다는 분석에 강함
  3. 🎨 이미지 생성 안 됨: 그림 그려주는 기능 없음
  4. 🌐 영어보다 중국어에 최적화: 중국 웹사이트 검색이 더 잘됨
챗GPT와의 차이점: 챗GPT는 "코딩의 신"이라면, 김 AI는 "문서의 신"입니다.
  • 코딩 과제 → 챗GPT가 더 정확
  • 보고서 100쪽 읽기 → 김 AI가 압도적
  • 한국어 대화 → 김 AI가 더 자연스러움
  • 최신 뉴스 분석 → 둘 다 가능하지만 김 AI가 검색을 더 많이 함
돈 없는 학생/직장인 → 김 AI가 답
프로그래머 → 챗GPT가 약간 더 나음
마누스 AI와의 차이점: 마누스 AI는 "AI 비서"라면, 김 AI는 "AI 전문가"입니다.
  • 마누스: "주식 분석하라"고 하면 직접 네이버 금융 가서 차트 그려줌
  • 김 AI: "주식 분석하라"고 하면 이론과 데이터를 정리해서 분석 리포트 써줌
마누스는 아직 초대장 필요고, 한국어 지원이 약합니다. 왕초보에게는 김 AI가 훨씬 접근성이 좋아요!

🎮 섹션 4: 누가? 무엇을? 어떻게? 얻으려면?

단계 1: 김 AI 시작하기 (1분 완성)

PC에서:
  1. 크롬 브라우저 켜기
  2. 주소창에 kimi.ai 치고 엔터
  3. "지금 시작하기" 또는 "로그인" 클릭
  4. 구글 계정으로 로그인 (또는 전화번호)
  5. 끝!
스마트폰에서:
  1. 앱스토어(iPhone) 또는 플레이스토어(갤럭시) 켜기
  2. "Kimi" 검색
  3. "Kimi - AI Assistant" 앱 다운로드
  4. 앱 켜서 구글 계정으로 로그인
  5. 끝!
왕초보 팁: 로그인 안 해도 쓸 수 있지만, 로그인하면 대화 기록 저장됩니다. 나중에 "아까 그거 다시 보여줘" 할 수 있어요!

단계 2: 내가 뭘 얻고 싶은지 정하기

김 AI에게 물어보기 전에 이렇게 생각해보세요:

"나는 [누구]이고, [무엇을] 얻고 싶어. 어떻게 해야 돼?"
🤔
누구 무엇을 얻고 싶은지 어떻게 할지 (김 AI에게 물어볼 말)
고등학생 수학 문제 풀이 방법 "이 수학 문제 풀어줘. 10번 문제인데, 고등학교 1학년 수준으로 쉽게 설명해줘"
대학생 논문 요약 "이 논문 30쪽 읽어주고, 핵심 3가지만 알려줘" + PDF 파일 올리기
직장인 회의록 작성 "이 회의 음성 파일 1시간 분량을 1쪽짜리 요약본으로 만들어줘"
주부 레시피 추천 "냉장고에 닭고기, 버섯, 양파 있는데 30분 안에 만들 수 있는 요리 추천해줘"
투자자 주식 분석 "삼성전자 재무제표 분석해줘. 2025년 11월 기준으로"
개발자 코드 오류 해결 "이 파이썬 코드 에러 뜨는데 왜 그런지 알려줘" + 코드 파일 올리기

"[무엇을] 하고 싶은데, [어떤 언어]로 코드 만들어줘. [누가] 쓸 거야" 예시: "주식 차트 그리고 싶은데, [파이썬]으로 코드 만들어줘. [코딩 처음 하는 내가] 이해할 수 있게 주석 달아줘"

💻📱 섹션 5: PC vs 스마트폰 답변 질 차이

결론: 답변 자체의 질은 거의 같습니다! 하지만 사용 경험과 속도는 차이가 납니다.

PC의 장점 💻

  1. 화면 커서 긴 답변 보기 편함
  2. 파일 드래그 앤 드롭 쉬움 (마우스로 끌어다 놓기)
  3. 코드 복사하기 편함
  4. 여러 개 창 띄워놓고 비교하기 쉬움
  5. 키보드로 빨리 타이핑 가능

스마트폰의 장점 📱

  1. 언제 어디서나 쓸 수 있음
  2. 음성 입력으로 말해서 물어볼 수 있음
  3. 사진 찍어서 바로 분석 가능 (예: 공식 문제집 찍어서 풀이 요청)
  4. 알림 받을 수 있음
  5. 카톡처럼 편하게 대화 가능
왕초보 추천 사용법
처음 배울 때: PC로 시작 (화면 크고, 파일 올리기 쉬움)
익숙해지면: 스마트폰으로 이동하면서 쓰기 (생활화)
중요 팁: 둘 다 같은 계정으로 로그인하면 대화 기록 공유됩니다. PC에서 하던 거 스마트폰에서 이어서 할 수 있어요!

🎯 섹션 6: 왕초보 초등수준 따라하기 (STEP BY STEP)

예제 1: "유튜브 영상 요약" 따라하기

상황: "10분짜리 경제 강의 영상 but 시간 없어요!"
  1. 유튜브 앱 켜기
  2. 영상 밑에 "공유" 버튼 누르기 (화살표 모양)
  3. "링크 복사" 누르기
  4. 김 AI 앱 켜기
  5. 검색창 길게 누르기 → "붙여넣기" 누르기
  6. "이 영상 3줄로 요약해줘" 라고 한글로 치기
  7. 전송 버튼 누르기 (↗️ 모양)
  8. 10초 기다리기
  9. 요약본 나오면 복사해서 카톡으로 친구한테 공유하기
왕초보 팁: 처음엔 영상이 "자막 지원"인지 확인하세요. 자막 없으면 요약이 힘들 수 있어요!

예제 2: "PDF 보고서 분석" 따라하기

상황: "회사 보고서 30페이지 but 읽기 싫어요!"
  1. 카카오톡으로 받은 PDF 파일 저장하기
  2. 김 AI 웹사이트 (kimi.ai) 열기
  3. 화면 중간 "파일 올리기" 버튼 클릭
  4. "내 PC" 클릭 → 방금 저장한 PDF 더블클릭
  5. 파일 올라가면 아래에 "이 보고서 1쪽으로 요약해줘. CEO 보고용으로" 라고 치기
  6. 전송 버튼 누르기
  7. 김 AI가 분석하는 동안 커피 마시기 ☕
  8. 완성된 요약본 복사해서 워드文档에 붙여넣기
  9. 이름 저장하고 상사한테 메일 보내기
초등학생 팁: 파일 이름을 "한글" 로 하면 김 AI가 더 잘 이해해요! 예: "2025년_3분기_실적보고서.pdf"

예제 3: "코딩 처음인데 웹사이트 만들기" 따라하기

상황: "코딩 1도 모르는데 '나만의 홈페이지' 만들고 싶어요!"
  1. 김 AI에게 말하기:
    나는 코딩 1도 모르는 초등학생 6학년이야. 내 이름, 사진, 취미 소개하는 간단한 홈페이지 코드 만들어줘. 파란색 배경이면 좋겠어
  2. 김 AI가 HTML 코드를 만들어줌
  3. 코드 오른쪽 위 "복사" 버튼 누르기
  4. 메모장 열기 → 붙여넣기
  5. 저장할 때 → 파일 이름: "myweb.html" → "모든 파일" 로 저장
  6. 그 파일 더블클릭 → 웹 브라우저로 열기
  7. 내 홈페이지 완성! 🎉
왕초보 팁: 김 AI한테 "코드 복사해서 어디에 붙이면 돼?" 라고 추가로 물어보면 상세 가이드까지 줍니다!

예제 4: "주식 분석 처음" 따라하기

상황: "삼성전자 주식 사고 싶은데 뭐부터 봐야할지 모르겠어요!"
  1. 김 AI에게 물어보기:
    나는 주식 처음 배우는 40세 주부야. 삼성전자 주식 분석하려는데, 뭘 봐야할지 모르겠어. 5가지만 쉽게 알려줘
  2. 김 AI가 답변:
    • ① 재무제표 보는법
    • ② PER/PBR 뜻
    • ③ 최근 뉴스
    • ④ 위험 요소
    • ⑤ 시작 방법
  3. 따라서 실제로 네이버 금융 들어가서 찾아보기
  4. 모르는 용어 또 김 AI에게 물어보기
  5. 종이에 메모해서 냉장고에 붙이기
왕초보 팁: "모르면 또 물어봐도 돼요!" 김 AI는 같은 질문을 100번 해도 화내지 않습니다.

순서:

  1. 김 AI 키고 "내일 서울 날씨 어때?" 치기
  2. "비 올까?" 추가로 물어보기
  3. "우산 가져가야 할까?" 라고 의견 물어보기

순서:

  1. 수학 문제 종이에 쓰기
  2. 사진 찍어서 김 AI에게 올리기
  3. "이 문제 풀어줘. 6학년 수준으로 설명해줘" 라고 물어보기

❌ 섹션 7: 왕초보가 가장 많이 하는 실수 TOP 5

실수 순위 실수 내용 원인 해결 방법
1 "답변 질이 왜 낮아요?" 질문이 애매함 "이거 알려줘" → "이거 [구체적으로] 알려줘"
2 "왜 안 돼요?" 로그인 안 했거나, 파일 크기가 너무 큼 먼저 김 AI 로그인, 파일은 100MB 넘지 않게
3 "답변이 길어서 끊겼어요" 너무 긴 답변을 한 번에 요청 "1단계만 먼저 알려줘"라고 나눠서 물어보기
4 "틀린 정보 줬어요!" 김 AI가 인터넷에서 틀린 정보를 가져옴 "이 정보 출처 알려줘"라고 물어보고, 공식 사이트에서 다시 확인하기
5 "다음에 또 쓰려니까 기록이 없어요" 로그인 안 하고 씀 무조건 로그인하고 쓰기! 대화 기록이 클라우드에 저장됨
기억하세요! 김 AI는 도구이지 만능이 아닙니다. 항상 본인의 판단이 필요해요!

💡 섹션 9: 왕초보가 꼭 알아야 할 10가지 팁

  1. 1️⃣ "모른다"고 솔직하게 말하기
    ❌ "알려줘"
    ✅ "이거 처음이라 하나도 모르는데, 1부터 10까지 쉽게 알려줘"
  2. 2️⃣ "누구를 위한"지 명확히 말하기
    ❌ "요약해줘"
    ✅ "고등학생도 이해할 수 있게 요약해줘" / "CEO 보고용으로 요약해줘"
  3. 3️⃣ "어떻게" 정확히 말하기
    ❌ "코드 줘"
    ✅ "파이썬으로 주식 차트 코드 줘. 주석도 달아줘"
  4. 4️⃣ "길이" 정하기
    ❌ "글 써줘"
    ✅ "블로그 글 500자로 써줘"
  5. 5️⃣ 파일 올리기 전에 "질문" 미리 쓰기
    파일 올리고 "이거 뭐야?" 보다는 "이 파일에서 중요한 3가지 찾아줘"라고 먼저 쓰기
  6. 6️⃣ "계속 대화"하기
    김 AI는 기억력이 있어요. "아까 그거 다시 설명해줘"라고 하면 이어서 답해줍니다.
  7. 7️⃣ "반대 의견" 구하기
    "이 주식 살까?" 물어본 뒤 "반대로 생각하면 어떤 리스크가 있어?" 라고 물어보면 더 정확한 판단 가능
  8. 8️⃣ "표"나 "목록"으로 달라고 하기
    "이 내용 표로 정리해줘"라고 하면 보기 편해요
  9. 9️⃣ "속도" 조절하기
    긴 답변은 "천천히" 나오니까 기다리세요. "아직 분석 중이야" 라고 뜨면 절대 새로고침 하지 마세요!
  10. 🔟 "틀린 정보" 체크하기
    김 AI가 "이 정보는 2024년 기준입니다"라고 하면 "2025년 11월 현재는 어때?"라고 다시 물어보기
