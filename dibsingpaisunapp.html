Pydroid 3란?

Pydroid 3는 안드로이드 스마트폰과 태블릿에서 파이썬 프로그래밍을 할 수 있게 해주는 앱입니다. PC 없이도 파이썬을 배우고, 코드를 작성하고, 실행할 수 있어 이동 중에도 학습이 가능합니다.

초보자부터 전문가까지 모두 사용할 수 있도록 다양한 기능을 제공하며, 무료 버전으로도 충분히 학습이 가능합니다.

Pydroid 3 설치하기

앱 설치 방법

Google Play 스토어에서 Pydroid 3를 설치하는 방법입니다.

단계별 설치 방법:

  1. 스마트폰에서 Google Play 스토어 열기
  2. 검색창에 "Pydroid 3" 입력
  3. 검색 결과에서 Pydroid 3 선택
  4. "설치" 버튼 클릭
  5. 다운로드 및 설치 완료 후 앱 실행

무료 버전으로도 대부분의 기능을 사용할 수 있습니다. 프리미엄 버전은 추가 기능을 제공합니다.

2
첫 실행 및 설정

초기 설정 방법

처음 앱을 실행하면 필요한 설정을 해야 합니다.

초기 설정 단계:

  1. 앱 실행 후 권한 요청 허용
  2. "Install" 버튼 클릭하여 파이썬 설치
  3. 설치 완료까지 기다리기 (5-10분 소요)
  4. 설정 아이콘(톱니바퀴) 클릭
  5. 테마 설정 (라이트/다크 모드 선택)
  6. 글자 크기 조절
  7. 저장 위치 확인

이제 파이썬 코딩을 시작할 준비가 되었습니다!

3
새 파일 만들기

새 파이썬 파일 생성

새로운 파이썬 파일을 만드는 방법입니다.

파일 생성 방법:

  1. 메인 화면에서 "+" 아이콘 클릭
  2. "New file" 선택
  3. 파일 이름 입력 (예: hello.py)
  4. "CREATE" 버튼 클릭

파이썬 파일은 확장자로 .py를 사용합니다. 이제 빈 편집 화면이 나타납니다.

4
첫 코드 작성하기

Hello World 출력

가장 기본적인 파이썬 코드를 작성해봅시다.

# 첫 파이썬 프로그램
print("안녕하세요, Pydroid 3!")
print("이것은 나의 첫 파이썬 프로그램입니다.")
print("스마트폰에서 파이썬을 실행하다니 신기해요!")

작성 방법:

  1. 편집 화면에 위 코드 입력
  2. 키보드의 "Tab" 키로 들여쓰기
  3. 한글 입력은 키보드 전환으로 가능
  4. 오타에 주의하며 입력
5
코드 실행하기

작성한 코드 실행

작성한 파이썬 코드를 실행하는 방법입니다.

실행 방법:

  1. 작성한 코드 확인
  2. 상단 메뉴에서 "실행" 아이콘(▶) 클릭
  3. 또는 키보드의 "F5" 키 누르기
  4. 하단에 결과 창이 나타남

실행 결과가 하단 패널에 표시됩니다. 오류가 발생하면 오류 메시지를 확인하세요.

6
파일 저장하기

코드 저장 방법

작성한 코드를 저장하여 나중에 다시 사용할 수 있습니다.

저장 방법:

  1. 상단 메뉴에서 "저장" 아이콘(💾) 클릭
  2. 또는 키보드의 "Ctrl+S" 누르기
  3. 파일 이름이 이미 있다면 자동 저장
  4. 새로운 이름으로 저장하려면 "다른 이름으로 저장" 선택

파일은 기본적으로 /Pydroid/ 폴더에 저장됩니다. 파일 탐색기 앱에서도 확인 가능합니다.

7
파일 불러오기

저장된 파일 열기

이전에 저장한 파일을 불러와서 편집할 수 있습니다.

불러오기 방법:

  1. 메인 화면에서 "폴더" 아이콘 클릭
  2. 저장된 파일 목록 확인
  3. 열고 싶은 파일 선택
  4. 최근 파일은 메인 화면에 표시됨

외부 파일을 가져오려면 파일 탐색기 앱에서 .py 파일을 Pydroid 3로 열기를 선택하세요.

8
예제 코드 사용하기

내장 예제 활용

Pydroid 3에는 학습에 유용한 예제 코드가 포함되어 있습니다.

예제 사용 방법:

  1. 메인 화면에서 "Samples" 선택
  2. 카테고리별 예제 목록 확인
  3. 원하는 예제 선택 (예: beginner > hello.py)
  4. "Open" 클릭하여 코드 확인
  5. 실행 버튼으로 결과 확인

예제 코드를 수정해보면서 다양한 기능을 학습할 수 있습니다.

9
패키지 설치하기 (pip)

pip로 라이브러리 설치

numpy, matplotlib 등 유용한 파이썬 라이브러리를 설치할 수 있습니다.

설치 방법:

  1. 메인 화면에서 "Pip" 선택
  2. 검색창에 원하는 패키지 이름 입력
  3. 패키지 선택 후 "Install" 클릭
  4. 설치 완료까지 기다리기
  5. 코드에서 import로 불러와 사용
# 설치한 패키지 사용 예시
import numpy as np
import matplotlib.pyplot as plt

# numpy 사용
arr = np.array([1, 2, 3, 4, 5])
print("평균값:", np.mean(arr))

자동 완성 기능

코드 자동 완성 사용법

Pydroid 3는 코드 작성 시 자동 완성 기능을 제공합니다.

사용 방법:

  1. 코드 입력 중 키워드 일부 입력
  2. 하단에 제안 목록 나타남
  3. 방향키로 원하는 항목 선택
  4. Tab 키로 자동 완성
  5. 함수 입력 시 괄호와 매개변수도 자동 완성

이 기능을 사용하면 오타를 줄이고 빠르게 코드를 작성할 수 있습니다.
