개발자도구 F12 링크 수정 연습

연습 방법:

  1. F12 키를 눌러 개발자도구를 실행하세요
  2. 왼쪽 상단의 화살표 아이콘을 클릭하세요
  3. 수정하고 싶은 링크를 클릭하여 선택하세요
  4. Elements 패널에서 링크 텍스트를 더블클릭하여 제목을 변경하세요
  5. href 속성을 더블클릭하여 링크 주소를 변경하세요
  6. Enter 키를 눌러 변경사항을 적용하세요

고급 연습:

  1. 링크에 target="_blank" 속성을 추가하여 새 창에서 열리도록 하세요
  2. 우클릭 > Edit as HTML로 전체 HTML을 편집해보세요
  3. Sources > Overrides를 사용하여 변경사항을 유지해보세요
  4. 링크에 title 속성을 추가하여 마우스 오버 시 툴팁을 표시해보세요
