시리즈 2편: AI 모델과의 대화, ChatGPT & Gemini 활용법

AI 모델이란? LLM이란?

생성형 AI의 원리 간단히 이해하기

생성형 AI(Generative AI)는 기존의 데이터를 학습하여 새로운 콘텐츠를 생성하는 인공지능 기술입니다. 이는 단순히 정보를 분류하거나 예측하는 전통적인 AI와는 차별화되는 개념으로, 텍스트, 이미지, 음악, 코드 등 다양한 형태의 창작물을 만들어낼 수 있습니다.

생성형 AI의 핵심은 방대한 양의 데이터를 통해 패턴과 구조를 학습하고, 이를 바탕으로 원래 학습 데이터와 유사하면서도 새로운 조합의 결과물을 출력하는 것입니다. 예를 들어, 수많은 문장을 학습한 AI는 단어와 단어 사이의 관계, 문법 구조, 문맥을 파악하여 주어진 프롬프트에 맞는 자연스러운 문장을 생성할 수 있습니다.

ChatGPT와 Google Gemini는 현재 대표적인 생성형 AI 모델로, 모두 대규모 언어 모델(LLM)에 기반하지만 각각의 개발사와 학습 방식, 강점에 따라 차이를 보입니다.

표 1: ChatGPT와 Gemini의 주요 특징 비교

ChatGPT를 효과적으로 사용하는 방법 (프롬프트 엔지니어링)

1

기본적인 프롬프트 작성 원칙

프롬프트 엔지니어링(Prompt Engineering)은 생성형 AI에게 원하는 답변을 얻기 위해 입력하는 질문이나 지시문(프롬프트)을 설계하는 기술입니다.

  • 명확성과 구체성: "주식에 대해 알려줘"보다 "삼성전자의 2024년 2분기 실적 분석해줘"
  • 맥락 제공: "나는 10년 이상의 장기 투자자입니다. 이 종목의 성장 가능성을 분석해줘"
  • 원하는 형식 지정: "표 형식으로 정리해줘", "JSON 형식으로 출력해줘"
  • 반복과 개선: 첫 답변을 보고 프롬프트를 수정하며 개선
2

역할 부여하기

AI에게 특정 역할(Role)을 부여하는 것은 프롬프트 엔지니어링에서 매우 강력한 기법입니다.

"당신은 10년 경력의 베테랑 주식 애널리스트입니다. 투자자들에게 전략적 조언을 제공하는 것이 당신의 임무입니다."

이를 통해 AI는 단순히 정보를 나열하는 것이 아니라, 투자 전략, 리스크 관리, 시장 심리 등을 고려한 전문적인 분석을 제공하려고 시도합니다.

3

출력 형식 지정하기

AI에게 원하는 출력 형식을 명시적으로 지정하는 것은, 생성된 결과를 프로그래밍적으로 활용하거나, 구조화된 정보를 얻기 위해 매우 중요한 기법입니다.

{ "company_name": "삼성전자", "financial_statement": { "revenue": "279.6조 원", "operating_income": "35.0조 원", "net_income": "25.1조 원" } }
4

예시를 통한 학습 (Few-shot Prompting)

Few-shot Prompting은 AI에게 원하는 작업을 직접 보여줌으로써, 더 정확하고 원하는 형태의 결과를 얻는 고급 프롬프트 엔지니어링 기법입니다.

뉴스: "삼성전자, 2분기 실적 기대치 상회...주가 상승"

감정: 긍정

뉴스: "현대차, 글로벌 공급망 차질로 생산 차질 우려"

감정: 부정

Google Gemini AI 활용하기

1

Google AI Studio 소개

Google AI Studio는 개발자들이 Google의 최신 AI 모델, 특히 Gemini 시리즈를 쉽게 테스트하고, 프로토타입을 만들며, API로 통합할 수 있도록 설계된 웹 기반 개발 도구입니다.

  • 프롬프트 실험실(Prompt Lab)을 통한 실시간 테스트
  • 모델 설정 조정 (응답 길이, 온도, 토큰 제한 등)
  • API 키 생성 및 관리 기능
  • SDK 및 코드 스니펫 제공
2

Gemini API 키 발급받기

  1. Google Cloud Console 접속: console.cloud.google.com
  2. 프로젝트 생성 또는 선택: 콘솔 상단의 프로젝트 드롭다운 메뉴에서 "새 프로젝트" 클릭
  3. API 및 서비스 활성화: "API 및 서비스" > "라이브러리"에서 "Generative Language API" 검색 후 사용 설정
  4. API 키 생성: "API 및 서비스" > "사용자 인증 정보"에서 "API 키" 생성
  5. API 키 복사 및 안전하게 보관: 생성된 키를 안전한 곳에 저장
3

google-generativeai 라이브러리 설치와 기본 사용법

설치 방법:

pip install google-generativeai

기본 사용법:

import google.generativeai as genai # API 키 설정 GOOGLE_API_KEY = "YOUR_API_KEY_HERE" genai.configure(api_key=GOOGLE_API_KEY) # 모델 선택 model = genai.GenerativeModel('gemini-pro') # 프롬프트 작성 및 응답 생성 response = model.generate_content("삼성전자의 최신 뉴스를 요약해줘") print(response.text)

import streamlit as st # 사이드바에 입력 위젯 배치 with st.sidebar: st.header("분석 설정") stock_symbol = st.text_input("종목 티커를 입력하세요 (예: AAPL)", value="AAPL") analysis_period = st.slider("분석 기간 (일)", min_value=30, max_value=730, value=365) analyze_button = st.button("분석 시작", type="primary") # 메인 콘텐츠 영역 if analyze_button and stock_symbol: st.write(f"'{stock_symbol}' 종목을 분석합니다...")
2

입력받은 종목 코드로 Alpha Vantage API 호출

import requests import pandas as pd def get_stock_data(symbol, period=365): """ Alpha Vantage API에서 주식 데이터를 가져옵니다. """ url = 'https://www.alphavantage.co/query' params = { 'function': 'TIME_SERIES_DAILY', 'symbol': symbol, 'apikey': 'YOUR_API_KEY', 'outputsize': 'full' } response = requests.get(url, params=params) data = response.json() # JSON 데이터를 DataFrame으로 변환 df = pd.DataFrame.from_dict(data["Time Series (Daily)"], orient='index') df = df.astype(float) df.index = pd.to_datetime(df.index) return df.tail(period)
3

받아온 데이터를 Streamlit으로 시각화

import mplfinance as mpf # 데이터 가져오기 stock_data = get_stock_data(stock_symbol, analysis_period) if stock_data is not None: # 캔들스틱 차트 그리기 fig, ax = mpf.plot( stock_data, type='candle', style='charles', title=f'{stock_symbol} Candlestick Chart', ylabel='Price ($)', volume=True, returnfig=True ) st.pyplot(fig) else: st.error("데이터를 가져오는 데 실패했습니다.")

import streamlit as st import pandas as pd import matplotlib.pyplot as plt import mplfinance as mpf from data_fetcher import get_stock_data from ai_analyzer import analyze_with_ai # 페이지 설정 st.set_page_config(page_title="AI 주식 분석 대시보드", layout="wide") # 제목 st.title("🚀 AI 주식 분석 대시보드") # 사이드바에 입력 위젯 배치 with st.sidebar: st.header("분석 설정") stock_symbol = st.text_input("종목 티커를 입력하세요 (예: AAPL)", value="AAPL") analysis_period = st.slider("분석 기간 (일)", min_value=30, max_value=730, value=365) analyze_button = st.button("분석 시작", type="primary") # 메인 콘텐츠 영역 if analyze_button and stock_symbol: try: with st.spinner(f"'{stock_symbol}'의 데이터를 가져오는 중..."): stock_data = get_stock_data(stock_symbol, period=analysis_period) if stock_data is not None and not stock_data.empty: st.success("데이터를 성공적으로 가져왔습니다!") # 차트와 AI 분석을 두 개의 열로 나누어 표시 col1, col2 = st.columns([2, 1]) with col1: st.subheader(f"{stock_symbol} 주가 차트 (최근 {analysis_period}일)") # 캔들스틱 차트 그리기 fig, ax = mpf.plot( stock_data, type='candle', style='charles', title=f'{stock_symbol} Candlestick Chart', ylabel='Price ($)', volume=True, mav=(20, 50), returnfig=True ) st.pyplot(fig) with col2: st.subheader("AI 기술적 분석") with st.spinner("AI가 차트를 분석 중..."): ai_analysis = analyze_with_ai(stock_data, analysis_type="technical") st.markdown(ai_analysis) else: st.error("해당 종목의 데이터를 찾을 수 없습니다. 티커를 확인해주세요.") except Exception as e: st.error(f"분석 중 오류가 발생했습니다: {e}") else: st.info("사이드바에서 종목 티커를 입력하고 '분석 시작' 버튼을 클릭하세요.")

데이터 가져오기 모듈 ( data_fetcher.py) 코드 작성

import requests import pandas as pd from config import ALPHA_VANTAGE_API_KEY def get_stock_data(symbol, period=365): """ Alpha Vantage API에서 주식 데이터를 가져옵니다. """ if not ALPHA_VANTAGE_API_KEY: raise ValueError("Alpha Vantage API 키가 설정되지 않았습니다. config.py를 확인하세요.") url = 'https://www.alphavantage.co/query' params = { 'function': 'TIME_SERIES_DAILY', 'symbol': symbol, 'apikey': ALPHA_VANTAGE_API_KEY, 'outputsize': 'full' # 최대 20년간의 데이터 } try: response = requests.get(url, params=params) response.raise_for_status() # HTTP 오류 발생 시 예외 발생 data = response.json() # API가 오류 메시지를 반환하는 경우 처리 if "Time Series (Daily)" not in data: print(f"API Error: {data}") return None # JSON 데이터를 DataFrame으로 변환 df = pd.DataFrame.from_dict(data["Time Series (Daily)"], orient='index') df = df.astype(float) df.index = pd.to_datetime(df.index) df = df.sort_index(ascending=True) # 컬럼명을 더 직관적으로 변경 df.rename(columns={ '1. open': 'Open', '2. high': 'High', '3. low': 'Low', '4. close': 'Close', '5. volume': 'Volume' }, inplace=True) # 요청한 기간만큼의 데이터만 반환 return df.tail(period) except requests.exceptions.RequestException as e: print(f"네트워크 오류 발생: {e}") return None except Exception as e: print(f"데이터 처리 중 오류 발생: {e}") return None

import google.generativeai as genai import pandas as pd from config import GOOGLE_GEMINI_API_KEY # Gemini API 초기화 genai.configure(api_key=GOOGLE_GEMINI_API_KEY) model = genai.GenerativeModel('gemini-pro') def analyze_with_ai(data: pd.DataFrame, analysis_type="technical"): """ 주가 데이터를 바탕으로 AI 분석을 수행합니다. """ if not GOOGLE_GEMINI_API_KEY: return "Google Gemini API 키가 설정되지 않았습니다. config.py를 확인하세요." try: if analysis_type == "technical": # DataFrame을 문자열로 변환하여 프롬프트에 포함 data_str = data.tail(30).to_string() # 최근 30일 데이터만 분석에 사용 prompt = f""" 당신은 전문 주식 애널리스트입니다. 다음은 특정 주식의 최근 일별 주가 데이터입니다. 데이터: {data_str} 위 데이터를 바탕으로 다음 항목에 대해 분석해주세요: 1. **최근 30일간의 추세**: 주가의 전반적인 방향성(상승, 하락, 횡보)을 판단하고, 그 근거를 설명해주세요. 2. **주요 지지/저항선**: 데이터를 바탕으로 예상되는 주요 지지선과 저항선 수준을 제시해주세요. 3. **투자 의견**: 위 분석을 바탕으로 향후 1-2주간의 투자 의견(매수, 매도, 보류)을 제시하고, 그 이유를 간단히 설명해주세요. 분석 결과는 마크다운 형식으로 작성해주세요. """ response = model.generate_content(prompt) return response.text # 다른 분석 유형 (예: 뉴스 분석)도 이곳에 추가할 수 있습니다. # elif analysis_type == "news": # ... except Exception as e: return f"AI 분석 중 오류가 발생했습니다: {e}"
