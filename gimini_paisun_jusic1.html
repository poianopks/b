파이썬과 주식 투자

왕초보를 위한 따라하기 공부 가이드

1. 왜 파이썬으로 주식 분석을 할까요?

주식 투자는 데이터와의 싸움입니다. 파이썬을 이용하면 이 싸움에서 강력한 무기를 얻을 수 있습니다.

  • 자동화: 매일 반복해야 하는 데이터 수집, 분석 작업을 자동으로 처리하여 시간을 절약합니다.
  • 방대한 데이터 처리: 수십 년 치의 주가 데이터도 파이썬을 이용하면 순식간에 처리하고 분석할 수 있습니다.
  • 객관적인 분석: 감정에 휘둘리지 않고, 데이터를 기반으로 한 전략을 세우고 검증(백테스팅)할 수 있습니다.
  • 강력한 생태계: 주식 분석에 필요한 수많은 도구(라이브러리)들이 이미 만들어져 있어, 우리는 가져다 쓰기만 하면 됩니다.

2. 무엇을 준비해야 할까요?

본격적인 분석에 앞서 컴퓨터에 몇 가지 프로그램을 설치해야 합니다.

  1. 파이썬 설치: 공식 홈페이지에서 최신 버전을 다운로드하여 설치하세요. 설치 시 'Add Python to PATH' 옵션을 꼭 체크해주세요.
  2. 코드 편집기 설치: 파이썬 코드를 작성하고 실행할 프로그램입니다. 초보자에게는 Visual Studio Code (VS Code)를 추천합니다.
  3. 필수 라이브러리 설치: '라이브러리'는 유용한 기능들을 모아놓은 코드 묶음입니다. 컴퓨터의 '터미널' 또는 '명령 프롬프트'를 열고 아래 명령어를 한 줄씩 입력하여 설치하세요. 
    pip install pandas matplotlib yfinance
    • pandas: 엑셀처럼 데이터를 표 형태로 다루는 도구
    • matplotlib: 데이터를 차트나 그래프로 그리는 도구
    • yfinance: 야후 파이낸스에서 주가 데이터를 가져오는 도구

3. 실전! 주가 데이터 가져오기

이제 파이썬 코드로 직접 주가 데이터를 가져와 보겠습니다. VS Code에서 새 파일(예: `stock_test.py`)을 만들고 아래 코드를 붙여넣은 후 실행해보세요.

여기서는 예시로 '삼성전자'의 데이터를 가져옵니다. 한국 주식은 종목코드 뒤에 .KS를 붙여야 합니다.

# 1. 필요한 라이브러리를 불러옵니다.
import yfinance as yf
import pandas as pd

# 2. 삼성전자(005930.KS)의 주가 데이터를 2023년부터 현재까지 가져옵니다.
samsung_df = yf.download('005930.KS', start='2023-01-01')

# 3. 데이터의 첫 5줄을 화면에 출력합니다.
print(samsung_df.head())

4. 데이터 시각화: 차트 그리기

숫자만 봐서는 흐름을 알기 어렵습니다. 데이터를 차트로 그려보면 주가의 움직임을 한눈에 파악할 수 있습니다. 위 코드에 이어서 아래 코드를 추가하고 실행해보세요.

# 4. 차트를 그리기 위한 라이브러리를 불러옵니다.
import matplotlib.pyplot as plt

# 5. 종가(Close) 데이터만 사용하여 차트를 그립니다.
plt.figure(figsize=(12, 6)) # 차트 크기 설정
plt.plot(samsung_df.index, samsung_df['Close'], label='Samsung Close Price')

plt.title('Samsung Electronics Stock Price') # 차트 제목
plt.xlabel('Date') # x축 라벨
plt.ylabel('Price (KRW)') # y축 라벨
plt.legend() # 범례 표시
plt.grid(True) # 그리드 표시
plt.show() # 차트 보여주기

코드를 실행하면 삼성전자 주가 흐름이 그려진 별도의 차트 창이 나타납니다. 이를 통해 언제 주가가 오르고 내렸는지 쉽게 확인할 수 있습니다.

5. 간단한 기술적 분석: 이동평균선

'이동평균선(Moving Average)'은 가장 기본적이면서도 중요한 기술적 분석 지표입니다. 일정 기간 동안의 주가 평균을 선으로 이은 것으로, 주가의 전반적인 추세를 파악하는 데 도움을 줍니다.

예를 들어, 20일 이동평균선은 최근 20일간의 종가 평균을 계산한 값입니다. pandas를 이용하면 아주 쉽게 계산할 수 있습니다.

# 6. 20일 이동평균선과 60일 이동평균선을 계산하여 새로운 열에 추가합니다.
samsung_df['MA20'] = samsung_df['Close'].rolling(window=20).mean()
samsung_df['MA60'] = samsung_df['Close'].rolling(window=60).mean()

# 7. 주가와 이동평균선을 함께 그립니다.
plt.figure(figsize=(12, 6))
plt.plot(samsung_df.index, samsung_df['Close'], label='Close Price', alpha=0.7)
plt.plot(samsung_df.index, samsung_df['MA20'], label='20-Day MA', linestyle='--')
plt.plot(samsung_df.index, samsung_df['MA60'], label='60-Day MA', linestyle=':')

plt.title('Samsung Electronics Price with Moving Averages')
plt.xlabel('Date')
plt.ylabel('Price (KRW)')
plt.legend()
plt.grid(True)
plt.show()

분석 팁:

  • 골든 크로스: 단기 이동평균선(20일)이 장기 이동평균선(60일)을 아래에서 위로 뚫고 올라가는 시점. 보통 강세 신호로 해석됩니다.
  • 데드 크로스: 단기 이동평균선이 장기 이동평균선을 위에서 아래로 뚫고 내려가는 시점. 보통 약세 신호로 해석됩니다.
