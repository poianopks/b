30개 주제로 체계적으로 배우는 파이썬 프로그래밍

스마트폰에서 Pydroid 3 앱을 사용하면 파이썬 코드를 쉽게 작성하고 실행할 수 있습니다. 각 주제를 클릭하여 상세 설명과 예제 코드를 확인하고, Pydroid 3에서 직접 따라해보세요!

Pydroid 3란?

Pydroid 3는 안드로이드 기기에서 파이썬 프로그래밍을 할 수 있게 해주는 앱입니다. PC 없이 스마트폰만으로 파이썬 코드를 작성하고 실행할 수 있어 언제 어디서나 학습이 가능합니다.

기본적인 코드 편집 기능부터 라이브러리 설치, 코드 실행 결과 확인까지 모든 기능을 제공합니다.

주요 기능

  • 파이썬 3 인터프리터 내장
  • 코드 자동 완성 및 하이라이트
  • pip를 통한 라이브러리 설치
  • 즉시 코드 실행 및 결과 확인
  • 예제 코드 템플릿 제공
  • 파일 관리 기능
1
변수(Variables)

변수

변수는 데이터를 저장하는 공간입니다. 파이썬에서는 변수를 선언할 때 데이터 타입을 명시하지 않습니다.

# 변수 선언 및 사용 예제
name = "김파이" # 문자열 변수
age = 10 # 정수형 변수
height = 150.5 # 실수형 변수
is_student = True # 불리언 변수

print("이름:", name)
print("나이:", age)
print("키:", height)
print("학생인가요?", is_student)
2
데이터 타입(Data Types)

데이터 타입

파이썬에는 여러 가지 기본 데이터 타입이 있습니다. 각 타입은 특정 종류의 데이터를 저장합니다.

# 주요 데이터 타입 예제
num_int = 42 # 정수
num_float = 3.14 # 실수
text = "Hello Python" # 문자열
is_true = True # 불리언
fruits = ["사과", "바나나", "오렌지"] # 리스트

print(type(num_int))
print(type(num_float))
print(type(text))
print(type(is_true))
print(type(fruits))
3
연산자(Operators)

연산자

파이썬에는 산술, 비교, 논리 연산자 등 다양한 연산자가 있습니다.

# 기본 연산자 예제
a = 10
b = 3

# 산술 연산자
print("a + b =", a + b) # 덧셈
print("a - b =", a - b) # 뺄셈
print("a * b =", a * b) # 곱셈
print("a / b =", a / b) # 나눗셈
print("a % b =", a % b) # 나머지
print("a ** b =", a ** b) # 제곱

# 비교 연산자
print("a == b?", a == b)
print("a > b?", a > b)
4
조건문(if문)

조건문

조건문은 특정 조건에 따라 다른 코드를 실행할 수 있게 해줍니다.

# if문 예제
score = 85

if score >= 90:
    print("A 학점입니다!")
elif score >= 80:
    print("B 학점입니다!")
elif score >= 70:
    print("C 학점입니다!")
else:
    print("D 학점입니다.")
5
반복문(for문)

for 반복문

for문은 컬렉션(리스트, 튜플, 문자열 등)의 요소들을 순회하며 코드를 반복 실행합니다.

# for문 예제
fruits = ["사과", "바나나", "딸기", "오렌지"]

# 리스트 순회
for fruit in fruits:
    print("과일:", fruit)

# range() 함수 사용
for i in range(5):
    print("숫자:", i)
6
반복문(while문)

while 반복문

while문은 조건이 참인 동안 코드를 반복 실행합니다.

# while문 예제
count = 0
# 5까지 세기
while count < 5:
    print("카운트:", count)
    count += 1 # count = count + 1

# 무한 루프와 break
while True:
    user_input = input("종료하려면 'q' 입력: ")
    if user_input == 'q':
        break
7
리스트(Lists)

리스트

리스트는 여러 값을 순서대로 저장하는 컬렉션 타입입니다. 변경 가능하며 다양한 데이터 타입을 저장할 수 있습니다.

# 리스트 예제
numbers = [1, 2, 3, 4, 5]
mixed = [10, "문자열", 3.14, True]

# 요소 접근
print("첫 번째 요소:", numbers[0])
print("마지막 요소:", numbers[-1])

# 리스트 조작
numbers.append(6) # 추가
numbers.insert(2, 2.5) # 삽입
del numbers[0] # 삭제
print("수정된 리스트:", numbers)

튜플(Tuples)

튜플

튜플은 리스트와 유사하지만 변경할 수 없는(immutable) 컬렉션 타입입니다.

# 튜플 예제
colors = ("빨강", "주황", "노랑", "초록")
single_tuple = (42,) # 요소가 하나일 때 쉼표 필요

# 요소 접근
print("두 번째 색상:", colors[1])

# 튜플은 변경 불가능
# colors[0] = "분홍" # 에러 발생

# 튜플 언패킹
r, o, y, g = colors
print("첫 번째 색상:", r)

딕셔너리(Dictionaries)

딕셔너리

딕셔너리는 키-값 쌍으로 데이터를 저장하는 컬렉션 타입입니다.

# 딕셔너리 예제
student = {
    "name": "김파이",
    "age": 20,
    "major": "컴퓨터 과학"
}

# 값 접근
print("이름:", student["name"])
print("나이:", student.get("age"))

# 값 변경 및 추가
student["age"] = 21 # 변경
student["gpa"] = 3.8 # 추가

# 키 순회
for key in student:
    print(key, ":", student[key])
10
집합(Sets)

집합

집합은 중복을 허용하지 않고 순서가 없는 컬렉션 타입입니다.

# 집합 예제
fruits = {"사과", "바나나", "오렌지", "사과"}
print(fruits) # {'사과', '바나나', '오렌지'}

# 집합 연산
setA = {1, 2, 3, 4, 5}
setB = {4, 5, 6, 7, 8}

print("합집합:", setA | setB)
print("교집합:", setA & setB)
print("차집합:", setA - setB)

Pydroid 3 설치하기

앱 설치 방법

Google Play 스토어에서 Pydroid 3를 설치하는 방법입니다.

단계별 설치 방법:

  1. 스마트폰에서 Google Play 스토어 열기
  2. 검색창에 "Pydroid 3" 입력
  3. 검색 결과에서 Pydroid 3 선택

무료 버전으로도 대부분의 기능을 사용할 수 있습니다. 프리미엄 버전은 추가 기능을 제공합니다.

첫 실행 및 설정

초기 설정 방법

처음 앱을 실행하면 필요한 설정을 해야 합니다.

초기 설정 단계:

새 파일 만들기

새 파이썬 파일 생성

새로운 파이썬 파일을 만드는 방법입니다.

파일 생성 방법:

  1. 메인 화면에서 "+" 아이콘 클릭

  4. "CREATE" 버튼 클릭

파이썬 파일은 확장자로 .py를 사용합니다. 이제 빈 편집 화면이 나타납니다.

코드 실행하기

작성한 코드 실행

작성한 파이썬 코드를 실행하는 방법입니다.

실행 방법:

  1. 작성한 코드 확인
  2. 상단 메뉴에서 "실행" 아이콘(▶) 클릭
  3. 또는 키보드의 "F5" 키 누르기
  4. 하단에 결과 창이 나타남

실행 결과가 하단 패널에 표시됩니다. 오류가 발생하면 오류 메시지를 확인하세요.

파일 저장하기

코드 저장 방법

작성한 코드를 저장하여 나중에 다시 사용할 수 있습니다.

저장 방법:

  1. 상단 메뉴에서 "저장" 아이콘(💾) 클릭
  2. 또는 키보드의 "Ctrl+S" 누르기
  3. 파일 이름이 이미 있다면 자동 저장
  4. 새로운 이름으로 저장하려면 "다른 이름으로 저장" 선택

파일은 기본적으로 /Pydroid/ 폴더에 저장됩니다. 파일 탐색기 앱에서도 확인 가능합니다.

파일 불러오기

저장된 파일 열기

이전에 저장한 파일을 불러와서 편집할 수 있습니다.

불러오기 방법:

  4. 최근 파일은 메인 화면에 표시됨

패키지 설치하기 (pip)

pip로 라이브러리 설치

numpy, matplotlib 등 유용한 파이썬 라이브러리를 설치할 수 있습니다.

설치 방법:

자동 완성 기능

코드 자동 완성 사용법

Pydroid 3는 코드 작성 시 자동 완성 기능을 제공합니다.

사용 방법:

  1. 코드 입력 중 키워드 일부 입력

  4. Tab 키로 자동 완성
  5. 함수 입력 시 괄호와 매개변수도 자동 완성

이 기능을 사용하면 오타를 줄이고 빠르게 코드를 작성할 수 있습니다.
