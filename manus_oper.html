💻 PC 왕초보를 위한
Browser Operator 완전 따라하기

PC를 켜는 것부터 Browser Operator로 일을 시키는 것까지
한 걸음씩 천천히 따라해보세요!

⏱️ 예상 소요 시간: 30분~1시간

🎯 이 가이드를 마치면 할 수 있는 것

  • ✅ Browser Operator를 PC에 설치하고 연결하기
  • ✅ 네이버, 구글 등 웹사이트에 자동으로 로그인하기
  • ✅ AI에게 "네이버에서 삼성전자 주가 찾아줘" 같은 명령하기
  • ✅ 복잡한 웹 작업을 AI가 대신하게 만들기

⚠️ 시작하기 전에 준비할 것

  • PC (컴퓨터) - 윈도우 10 이상 또는 맥OS
  • 인터넷 연결 - 와이파이 또는 유선
  • 이메일 주소 - Manus 계정 만들기용
  • 30분~1시간의 시간 - 천천히 따라하세요

📚 전체 과정 미리보기

단계 할 일 소요 시간
1단계 PC 켜고 Chrome 브라우저 열기 5분
2단계 Manus 계정 만들기 5분
3단계 Browser Operator 확장 프로그램 설치 5분
4단계 Manus와 Browser Operator 연결 5분
5단계 네이버에 로그인하기 5분
6단계 Browser Operator 작동 테스트 5분

🚀 1단계: PC 켜고 Chrome 브라우저 열기

1-1

PC 전원 켜기

PC의 전원 버튼을 눌러서 컴퓨터를 켜세요.

💡 팁: 전원 버튼은 보통 PC 본체 앞면이나 노트북 키보드 위쪽에 있어요. 동그란 모양이거나 전원 표시(⏻)가 그려져 있어요.

기다리기: PC가 완전히 켜질 때까지 1~2분 정도 기다려주세요. 윈도우 로고나 로그인 화면이 나타나면 준비된 거예요!

1-2

윈도우에 로그인하기

비밀번호를 입력하거나 PIN 번호를 입력해서 윈도우에 로그인하세요.

① 비밀번호 입력: 키보드로 비밀번호를 천천히 입력하세요.
② Enter 누르기: 키보드에서 Enter 키를 눌러주세요.
💡 팁: 비밀번호를 잊어버렸다면? 화면 아래에 "비밀번호를 잊으셨나요?" 또는 "암호 재설정" 버튼을 찾아서 클릭하세요.
1-3

Chrome 브라우저가 있는지 확인하기

Chrome 브라우저가 이미 설치되어 있는지 확인해볼게요.

① 바탕화면 확인: 바탕화면에 Chrome 아이콘(빨강/노랑/초록/파랑 동그라미)이 있나요?
② 있으면: 더블클릭 (빠르게 두 번 클릭)해서 Chrome을 열어주세요.
③ 없으면: 다음 단계(1-4)로 넘어가서 Chrome을 설치해주세요.
1-4

Chrome 브라우저 설치하기 (없는 경우만)

Chrome이 없다면 지금 설치해볼게요!

① Edge 브라우저 열기: 윈도우에는 기본적으로 Edge 브라우저가 설치되어 있어요. 화면 아래 작업 표시줄에서 파란색 "e" 모양 아이콘을 찾아서 클릭하세요.
② Chrome 다운로드 페이지 접속: 주소창(위쪽 긴 네모 칸)을 클릭하고, 다음 주소를 입력하세요:
https://www.google.com/chrome
입력 후 Enter 키를 누르세요.
③ "Chrome 다운로드" 버튼 클릭: 페이지가 열리면 큰 파란색 버튼에 "Chrome 다운로드"라고 써있을 거예요. 그 버튼을 클릭하세요.
④ 다운로드 기다리기: 파일이 다운로드될 때까지 30초~1분 정도 기다려주세요.
⑤ 설치 파일 실행: 다운로드가 끝나면 화면 아래쪽이나 왼쪽 아래에 "ChromeSetup.exe" 파일이 나타날 거예요. 그것을 클릭하세요.
⑥ 설치 진행: "예" 또는 "설치" 버튼이 나오면 클릭하세요. Chrome이 자동으로 설치됩니다. 2~3분 정도 기다려주세요.
⑦ Chrome 열기: 설치가 끝나면 Chrome이 자동으로 열릴 거예요!

✅ 1단계 완료!

이제 Chrome 브라우저가 열려 있나요? 그렇다면 다음 단계로 넘어갈 준비가 되었어요!

👤 2단계: Manus 계정 만들기

2-1

Manus 웹사이트 접속하기

① 주소창 클릭: Chrome 브라우저 위쪽에 있는 긴 네모 칸(주소창)을 클릭하세요.
② Manus 주소 입력: 다음 주소를 천천히 입력하세요:
https://manus.im
③ Enter 누르기: Enter 키를 눌러서 Manus 웹사이트로 이동하세요.
💡 팁: 주소를 입력할 때 대문자/소문자는 구분하지 않아도 돼요. "manus.im"이라고만 입력해도 자동으로 "https://"가 붙어요.
2-2

회원가입 시작하기

① "Sign Up" 또는 "회원가입" 버튼 찾기: Manus 웹사이트가 열리면 화면 오른쪽 위에 "Sign Up" 또는 "회원가입" 버튼이 있을 거예요.
② 버튼 클릭: 그 버튼을 클릭하세요.
2-3

정보 입력하기

① 이메일 주소 입력: 본인의 이메일 주소를 입력하세요. 예: yourname@gmail.com
② 비밀번호 만들기: 기억하기 쉬운 비밀번호를 만들어서 입력하세요.
⚠️ 비밀번호 규칙:
  • 8자 이상
  • 영문 대문자와 소문자 포함
  • 숫자 포함
  • 특수문자 포함 (예: !, @, #, $)
예시: MyPassword123!
③ 비밀번호 확인: 같은 비밀번호를 한 번 더 입력하세요.
④ "가입" 또는 "Sign Up" 버튼 클릭: 모든 정보를 입력했으면 버튼을 클릭하세요.
2-4

이메일 인증하기

① 이메일 확인: 방금 입력한 이메일 주소로 Manus에서 인증 이메일을 보냈을 거예요. 이메일을 확인하러 가세요.
② 새 탭 열기: Chrome에서 Ctrl + T 키를 동시에 눌러서 새 탭을 여세요.
③ 이메일 사이트 접속: Gmail을 사용한다면 "gmail.com"을 입력하고 Enter를 누르세요. 다른 이메일을 사용한다면 해당 사이트로 가세요.
④ Manus 이메일 찾기: 받은 편지함에서 "Manus" 또는 "manus.im"에서 온 이메일을 찾으세요.
⑤ 인증 링크 클릭: 이메일을 열고, 안에 있는 "이메일 인증" 또는 "Verify Email" 버튼을 클릭하세요.
💡 팁: 이메일이 안 보인다면? 스팸 메일함을 확인해보세요. 그래도 없다면 5분 정도 기다렸다가 다시 확인하세요.
2-5

Manus에 로그인하기

① Manus 탭으로 돌아가기: Chrome 위쪽에 있는 탭들 중에서 "Manus" 탭을 클릭하세요.
② 로그인 정보 입력: 이메일 주소와 비밀번호를 입력하세요.
③ "로그인" 또는 "Sign In" 버튼 클릭: 버튼을 클릭해서 Manus에 로그인하세요.

✅ 2단계 완료!

Manus 계정이 만들어졌고 로그인도 완료되었어요! 이제 Browser Operator를 설치할 준비가 되었습니다.

🔧 3단계: Browser Operator 확장 프로그램 설치

3-1

Browser Operator 페이지로 이동하기

① Manus 웹사이트에서 찾기: Manus에 로그인한 상태에서, 화면 위쪽 메뉴에서 "Browser Operator" 또는 "기능" 메뉴를 찾으세요.
② 또는 직접 접속: 새 탭을 열고 다음 주소를 입력하세요:
https://manus.im/features/manus-browser-operator
3-2

"Install Extension" 버튼 클릭하기

① 버튼 찾기: Browser Operator 페이지에서 "Install Extension" 또는 "확장 프로그램 설치" 버튼을 찾으세요. 보통 페이지 위쪽에 큰 버튼으로 있어요.
② 버튼 클릭: 그 버튼을 클릭하세요.
③ Chrome 웹 스토어로 이동: 자동으로 Chrome 웹 스토어 페이지가 열릴 거예요.
3-3

Chrome 웹 스토어에서 설치하기

① "Chrome에 추가" 버튼 클릭: Chrome 웹 스토어 페이지에서 파란색 "Chrome에 추가" 버튼을 찾아서 클릭하세요.
② 확인 팝업: "Manus Browser Operator를 추가하시겠습니까?"라는 작은 창이 나타날 거예요.
③ "확장 프로그램 추가" 클릭: 그 창에서 "확장 프로그램 추가" 버튼을 클릭하세요.
④ 설치 기다리기: 10~20초 정도 기다리면 설치가 완료됩니다.
3-4

설치 확인하기

① 확장 프로그램 아이콘 찾기: Chrome 오른쪽 위에 퍼즐 조각 모양 아이콘이 있어요. 그것을 클릭하세요.
② Manus Browser Operator 확인: 목록에서 "Manus Browser Operator"가 보이나요? 보인다면 설치 성공!
③ 고정하기 (선택사항): "Manus Browser Operator" 옆에 있는 핀 📌 아이콘을 클릭하면, Chrome 오른쪽 위에 항상 표시됩니다.

✅ 3단계 완료!

Browser Operator 확장 프로그램이 설치되었어요! 이제 Manus와 연결할 차례입니다.

🔗 4단계: Manus와 Browser Operator 연결

4-1

Manus 채팅 페이지 열기

① 새 탭 열기: Ctrl + T 키를 동시에 눌러서 새 탭을 여세요.
② Manus 채팅 접속: 주소창에 다음을 입력하세요:
https://manus.im/chat
③ Enter 누르기: Enter 키를 눌러서 Manus 채팅 페이지로 이동하세요.
4-2

Browser Operator 연결 시작하기

① 채팅창 확인: Manus 채팅 페이지가 열리면, 화면 중앙에 채팅창이 보일 거예요.
② Browser Operator 아이콘 클릭: Chrome 오른쪽 위에 있는 Manus Browser Operator 아이콘을 클릭하세요. (퍼즐 조각 아이콘을 먼저 클릭한 후 Manus를 찾아야 할 수도 있어요)
③ 작은 창 열림: Browser Operator 아이콘을 클릭하면 작은 창이 나타날 거예요.
④ "Connect" 또는 "연결" 버튼 클릭: 그 작은 창에서 "Connect" 또는 "연결" 버튼을 찾아서 클릭하세요.
4-3

연결 확인하기

① 연결 상태 확인: Browser Operator 아이콘을 다시 클릭해보세요.
② "Connected" 또는 "연결됨" 표시: 작은 창에 "Connected" 또는 "연결됨"이라고 표시되면 성공!
③ 초록색 불빛: Browser Operator 아이콘에 초록색 불빛이나 체크 표시가 나타나면 연결된 거예요.

✅ 4단계 완료!

Manus와 Browser Operator가 연결되었어요! 이제 Manus AI가 내 브라우저를 조종할 수 있습니다.

🔐 5단계: 네이버에 로그인하기

📌 왜 네이버에 로그인하나요?

Browser Operator를 테스트하기 위해 네이버에 로그인해둘 거예요. 나중에 Manus AI에게 "네이버에서 삼성전자 주가 찾아줘"라고 명령하면, AI가 이미 로그인된 네이버 계정을 사용해서 자동으로 찾아줍니다!

5-1

네이버 접속하기

① 새 탭 열기: Ctrl + T 키를 동시에 눌러서 새 탭을 여세요.
② 네이버 주소 입력: 주소창에 다음을 입력하세요:
https://www.naver.com
③ Enter 누르기: Enter 키를 눌러서 네이버로 이동하세요.
5-2

네이버 로그인하기

① "로그인" 버튼 찾기: 네이버 메인 페이지 오른쪽 위에 "로그인" 버튼이 있어요.
② "로그인" 클릭: 그 버튼을 클릭하세요.
③ 아이디 입력: 네이버 아이디를 입력하세요.
④ 비밀번호 입력: 네이버 비밀번호를 입력하세요.
⑤ "로그인" 버튼 클릭: 초록색 "로그인" 버튼을 클릭하세요.
💡 팁: 네이버 계정이 없다면? "회원가입" 버튼을 클릭해서 먼저 네이버 계정을 만드세요. 5분 정도 걸려요.
5-3

로그인 상태 유지하기

① "로그인 상태 유지" 체크: 로그인 페이지에 "로그인 상태 유지" 체크박스가 있다면 체크하세요. 이렇게 하면 다음에 네이버를 열 때 자동으로 로그인됩니다.
② 로그인 확인: 로그인이 완료되면 네이버 메인 페이지 오른쪽 위에 내 아이디가 표시될 거예요.

✅ 5단계 완료!

네이버에 로그인했어요! 이제 Manus AI가 이 로그인 상태를 사용해서 네이버에서 정보를 찾아줄 수 있습니다.

🎮 6단계: Browser Operator 작동 테스트

6-1

Manus 채팅 페이지로 돌아가기

① Manus 탭 찾기: Chrome 위쪽 탭들 중에서 "Manus" 또는 "manus.im/chat" 탭을 찾으세요.
② 탭 클릭: 그 탭을 클릭해서 Manus 채팅 페이지로 돌아가세요.
6-2

첫 번째 명령 내리기

① 채팅창 클릭: Manus 채팅 페이지 아래쪽에 있는 채팅 입력창을 클릭하세요.
② 명령 입력: 다음 명령을 입력하세요:
네이버에서 삼성전자 현재 주가를 찾아줘
③ Enter 누르기: Enter 키를 눌러서 명령을 보내세요.
6-3

AI가 일하는 모습 지켜보기

① Browser Operator 작동: 명령을 보내면, Manus AI가 Browser Operator를 통해 자동으로 작업을 시작합니다!
② 브라우저 자동 조작: Chrome 브라우저가 자동으로 움직이는 것을 볼 수 있어요:
  • 새 탭이 자동으로 열림
  • 네이버 검색창에 "삼성전자 주가"가 자동으로 입력됨
  • 검색 버튼이 자동으로 클릭됨
  • 주가 정보가 자동으로 수집됨
③ 결과 확인: 10~30초 후에 Manus 채팅창에 삼성전자 주가 정보가 나타날 거예요!
💡 팁: AI가 일하는 동안 브라우저를 만지지 마세요! AI가 작업을 완료할 때까지 기다려주세요.
6-4

더 많은 명령 시도해보기

이제 Browser Operator가 작동하는 것을 확인했으니, 다른 명령도 시도해보세요!

🎯 시도해볼 만한 명령 예시:

1. 날씨 확인:
네이버에서 서울 날씨를 찾아줘
2. 뉴스 검색:
네이버에서 "AI 뉴스" 최신 기사 3개를 찾아줘
3. 제품 검색:
네이버 쇼핑에서 "무선 이어폰" 인기 제품 5개를 찾아줘
4. 지도 검색:
네이버 지도에서 강남역 근처 맛집을 찾아줘

💡 Browser Operator ON 상태 유지하기

📌 Browser Operator를 계속 사용하려면?

Browser Operator는 Chrome 브라우저가 열려 있고, Manus에 로그인되어 있으면 자동으로 연결 상태를 유지합니다.

✅ ON 상태 확인 방법

① Browser Operator 아이콘 확인: Chrome 오른쪽 위에 있는 Manus Browser Operator 아이콘을 클릭하세요.
② 연결 상태 확인:
  • 초록색 불빛 또는 "Connected": ON 상태 ✅
  • 회색 불빛 또는 "Disconnected": OFF 상태 ❌
③ OFF 상태라면? "Connect" 또는 "연결" 버튼을 다시 클릭하세요.

🔄 다음에 PC를 켤 때

PC를 끄고 다음에 다시 켤 때는 다음 단계만 하면 돼요:

① Chrome 열기: Chrome 브라우저를 실행하세요.
② Manus 접속: manus.im/chat으로 접속하세요.
③ 자동 로그인: 이전에 "로그인 상태 유지"를 체크했다면 자동으로 로그인됩니다.
④ Browser Operator 자동 연결: 대부분의 경우 Browser Operator가 자동으로 연결됩니다. 안 되면 아이콘을 클릭해서 "Connect"를 누르세요.

🎓 Browser Operator 활용 팁

💡 효과적으로 사용하는 방법

1. 명확한 명령 내리기

좋은 예: "네이버에서 삼성전자 현재 주가와 전일 대비 변동률을 찾아줘"

나쁜 예: "주가 좀" (무엇을 찾아야 할지 불명확)

2. 단계별로 명령하기

복잡한 작업은 여러 단계로 나눠서 명령하세요.

예:

  1. 먼저: "네이버에서 삼성전자 주가를 찾아줘"
  2. 그 다음: "이제 SK하이닉스 주가도 찾아줘"
  3. 마지막: "두 회사의 주가를 비교해줘"

3. 로그인 상태 유지하기

자주 사용하는 웹사이트(네이버, 구글, 페이스북 등)는 미리 로그인해두세요. Browser Operator가 그 로그인 상태를 사용해서 더 많은 일을 할 수 있어요.

4. AI가 일하는 동안 기다리기

Browser Operator가 작동 중일 때는 브라우저를 만지지 마세요. AI가 작업을 완료할 때까지 기다려주세요.

❓ 자주 묻는 질문 (FAQ)

Q1: Browser Operator가 연결되지 않아요

해결 방법:

  1. Chrome을 완전히 종료하고 다시 여세요
  2. Manus에 다시 로그인하세요
  3. Browser Operator 아이콘을 클릭하고 "Connect" 버튼을 다시 누르세요
  4. 그래도 안 되면 확장 프로그램을 제거하고 다시 설치하세요

Q2: AI가 명령을 이해하지 못해요

해결 방법:

  1. 명령을 더 명확하게 다시 작성하세요
  2. "네이버에서", "구글에서" 같이 어디서 찾을지 명시하세요
  3. 한 번에 하나의 작업만 요청하세요

Q3: 브라우저가 자동으로 움직이지 않아요

해결 방법:

  1. Browser Operator 연결 상태를 확인하세요 (초록색 불빛)
  2. 로그인이 필요한 사이트라면 먼저 수동으로 로그인하세요
  3. 팝업 차단이 켜져 있다면 끄세요

Q4: 보안이 걱정돼요

안심하세요:

  • Browser Operator는 내 PC에서만 작동합니다
  • 내 로그인 정보는 내 브라우저에만 저장되고, Manus 서버로 전송되지 않습니다
  • 언제든지 Browser Operator를 끄거나 제거할 수 있습니다

Q5: 스마트폰에서도 사용할 수 있나요?

답변:

Browser Operator는 PC 전용입니다. 하지만 Manus AI의 다른 기능(Wide Research, 웹사이트 빌더 등)은 스마트폰에서도 사용할 수 있어요!
