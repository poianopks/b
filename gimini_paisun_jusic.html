기업 주식 공부로 파이썬 기초 배우기

주식 분석과 파이썬 코딩, 두 마리 토끼를 한 번에!

1. 왜 파이썬으로 주식 분석을 할까요?

주식 투자는 데이터와의 싸움입니다. 파이썬을 이용하면 이 싸움에서 강력한 무기를 얻을 수 있습니다.

  • 자동화: 매일 반복해야 하는 데이터 수집, 분석 작업을 자동으로 처리하여 시간을 절약합니다.
  • 방대한 데이터 처리: 수십 년 치의 주가 데이터도 파이썬을 이용하면 순식간에 처리하고 분석할 수 있습니다.
  • 객관적인 분석: 감정에 휘둘리지 않고, 데이터를 기반으로 한 전략을 세우고 검증(백테스팅)할 수 있습니다.
  • 강력한 생태계: 주식 분석에 필요한 수많은 도구(라이브러리)들이 이미 만들어져 있어, 우리는 가져다 쓰기만 하면 됩니다.

2. 무엇을 준비해야 할까요?

본격적인 분석에 앞서 컴퓨터에 몇 가지 프로그램을 설치해야 합니다.

  1. 파이썬 설치: 공식 홈페이지에서 최신 버전을 다운로드하여 설치하세요. 설치 시 'Add Python to PATH' 옵션을 꼭 체크해주세요.
  2. 코드 편집기 설치: 파이썬 코드를 작성하고 실행할 프로그램입니다. 초보자에게는 Visual Studio Code (VS Code)를 추천합니다.
  3. 필수 라이브러리 설치: '터미널' 또는 '명령 프롬프트'를 열고 아래 명령어를 한 줄씩 입력하여 설치하세요. 
    pip install pandas matplotlib yfinance beautifulsoup4 requests
    • pandas, matplotlib, yfinance: 이전 단계에서 설치
    • beautifulsoup4, requests: 웹사이트 정보를 가져오기 위한 라이브러리 (뉴스 크롤링용)

3. 실전! 주가 데이터 가져오기

파이썬 코드로 직접 주가 데이터를 가져와 보겠습니다. 삼성전자(종목코드: 005930)의 데이터를 예시로 사용합니다.

import yfinance as yf
samsung_df = yf.download('005930.KS', start='2023-01-01')
print(samsung_df.head())

4. 데이터 시각화: 차트 그리기

주가 데이터를 차트로 그려 움직임을 한눈에 파악합니다.

import matplotlib.pyplot as plt
plt.figure(figsize=(12, 6))
plt.plot(samsung_df.index, samsung_df['Close'])
plt.title('Samsung Electronics Stock Price')
plt.show()

5. 기술적 분석: 이동평균선

이동평균선을 계산하여 주가의 추세를 파악합니다.

samsung_df['MA20'] = samsung_df['Close'].rolling(window=20).mean()
samsung_df['MA60'] = samsung_df['Close'].rolling(window=60).mean()
plt.figure(figsize=(12, 6))
plt.plot(samsung_df.index, samsung_df['Close'], label='Close')
plt.plot(samsung_df.index, samsung_df['MA20'], label='MA20')
plt.plot(samsung_df.index, samsung_df['MA60'], label='MA60')
plt.legend()
plt.show()

6. 기업 분석: 재무제표 확인하기

기업의 건강 상태를 알려주는 재무제표를 파이썬으로 쉽게 가져올 수 있습니다. 이는 기업의 본질적인 가치를 분석하는 '기본적 분석'의 핵심입니다.

# 1. 분석할 기업의 Ticker 객체를 생성합니다.
ticker = yf.Ticker("005930.KS") # 삼성전자

# 2. 기업의 주요 재무 정보를 가져옵니다. (손익계산서)
financials = ticker.financials
print("--- 손익계산서 (연간) ---")
print(financials)

# 3. 기업의 대차대조표를 가져옵니다.
balance_sheet = ticker.balance_sheet
print("\n--- 대차대조표 (연간) ---")
print(balance_sheet)

# 4. 현금흐름표를 가져옵니다.
cash_flow = ticker.cashflow
print("\n--- 현금흐름표 (연간) ---")
print(cash_flow)

파이썬 기초 공부 포인트!

  • 객체(Object)와 메소드(Method): yf.Ticker("...")는 'Ticker 객체'를 만드는 코드입니다. 이 객체는 .financials, .balance_sheet 와 같은 자신만의 데이터(속성)와 기능(메소드)을 가지고 있습니다. 객체 지향 프로그래밍의 기본 개념을 맛볼 수 있습니다.
  • 데이터 구조 확인: 출력된 결과를 통해 복잡한 금융 데이터가 어떻게 '표(DataFrame)' 형태로 정리되는지 확인 할 수 있습니다.

7. 최신 정보: 기업 뉴스 제목 가져오기 (웹 크롤링 기초)

투자에 있어 최신 정보는 매우 중요합니다. 파이썬의 '웹 크롤링' 기술을 이용하면 온라인 뉴스를 자동으로 가져올 수 있습니다. 여기서는 `yfinance` 라이브러리가 제공하는 뉴스 기능을 활용하는 간단한 방법을 알아봅니다.

주의: 웹 크롤링은 웹사이트의 정책(robots.txt)을 반드시 존중해야 하며, 과도한 요청은 서버에 부담을 줄 수 있습니다. 상업적 이용이나 무단 배포는 법적 문제가 될 수 있으니 항상 주의하세요. 
# 1. 분석할 기업의 Ticker 객체를 생성합니다.
ticker = yf.Ticker("005930.KS") # 삼성전자

# 2. 해당 기업 관련 최신 뉴스를 가져옵니다.
news = ticker.news

# 3. 뉴스의 제목과 링크를 하나씩 출력합니다.
print("--- 삼성전자 최신 뉴스 ---")
for item in news:
    print(f"제목: {item['title']}")
    print(f"링크: {item['link']}\n")

파이썬 기초 공부 포인트!

  • 리스트(List)와 딕셔너리(Dictionary): ticker.news는 여러 개의 뉴스 정보를 담고 있는 '리스트'입니다. 각 뉴스는 '제목', '링크' 등의 정보를 '키(key)'와 '값(value)'으로 묶은 '딕셔너리' 형태로 되어 있습니다.
  • 반복문 (for loop): for item in news: 구문은 리스트에 있는 모든 뉴스를 하나씩 꺼내어 중괄호 안의 코드를 반복 실행합니다. 데이터 처리의 가장 기본이 되는 문법입니다.
  • 문자열 포매팅 (f-string): f"제목: {item['title']}" 처럼 문자열 앞에 'f'를 붙이면, 중괄호 안에 변수나 값을 넣어 쉽게 문자열을 조합할 수 있습니다.
