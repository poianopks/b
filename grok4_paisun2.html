파이썬이 뭐야? (쉬운 개념)

파이썬은 컴퓨터에게 "이거 해!"라고 말하는 쉬운 언어야. 마치 친구랑 이야기하듯 쓰여. 예를 들어, "안녕!"이라고 화면에 띄우려면:

print("안녕!")

이렇게 쓰면 돼! 파이썬은 게임 만들기, 계산하기, 웹사이트 만들기 등 재미있는 일에 써.

왜 쉬워? 영어처럼 쓰여서 읽기 편해. 초등학생도 배울 수 있어!

Flask가 뭐야? (쉬운 개념)

Flask는 파이썬으로 웹사이트를 빨리 만드는 도구야. 마치 레고 블록처럼 쉽게 쌓아. 복잡한 거 몰라도 돼. 예를 들어, 웹 페이지에 "환영해!"라고 띄우는 거야.

Flask는 작고 가벼워서 초보자가 좋아해. 큰 웹사이트도 만들 수 있지만, 처음엔 간단한 거부터!

Flask와 파이썬은 어떤 친구야? (쉬운 개념)

파이썬은 기본 언어, Flask는 그 언어로 웹을 만드는 특별한 도구야. 비유하자면, 파이썬은 색연필이고 Flask는 그 색연필로 그림 그리는 방법이야. Flask를 쓰려면 파이썬이 먼저 필요해!

파이썬과 Flask 설치하기 (1단계부터 따라하기)

컴퓨터에 파이썬이 없으면 먼저 설치하자. 초등학생처럼 천천히 따라 해!

  1. 인터넷 브라우저 열고 python.org 가서 "Download Python" 버튼 클릭. (최신 버전 다운로드)
  2. 다운로드된 파일 열고 설치. "Add Python to PATH" 체크하고 설치 누르기.
  3. 설치 끝나면, 컴퓨터 검색창에 "cmd" 쳐서 검은 창(명령 프롬프트) 열기.
  4. 검은 창에 python --version 쳐서 숫자 나오면 성공!
  5. 이제 Flask 설치: 검은 창에 pip install Flask 쳐서 기다리기. 끝나면 준비됐어!

문제 생기면 부모님께 물어보거나 인터넷 검색해!

파이썬 코드 작성하기 (웹사이트 뼈대 만들기)

이제 코드를 써보자. 메모장이나 VS Code(무료 프로그램) 열고 새 파일 만들어 "app.py"로 저장해.

파일 안에 이 코드를 복사해서 붙여넣어:

from flask import Flask app = Flask(__name__) @app.route('/') def home(): return "안녕! 내 첫 웹사이트야!" if __name__ == '__main__': app.run(debug=True)

이 코드는 웹 페이지에 "안녕!"이라고 보여주는 거야. 저장하고 나서 다음 단계로!

내 컴퓨터에서 테스트하기 (로컬 테스트)

검은 창(명령 프롬프트) 열고, app.py 파일 있는 폴더로 가. (cd 명령어 써서 폴더 이동)

  1. 검은 창에 python app.py 쳐서 실행.
  2. 브라우저 열고 주소창에 http://127.0.0.1:5000 쳐서 들어가.
  3. "안녕! 내 첫 웹사이트야!" 나오면 성공! 안 나오면 코드 다시 확인해.

이게 로컬 테스트야. 내 컴퓨터에서만 보이는 거지!

GitHub에 저장하기 (인터넷에 코드 보관하기)

GitHub는 코드 보관하는 인터넷 창고야. 무료로 쓸 수 있어!

  1. github.com 가서 계정 만들기 (이메일 필요).
  2. 로그인 후 "New repository" 버튼 클릭. 이름 지어주고 "Create repository" 누르기.
  3. 컴퓨터에 Git 설치 (검색해서 다운로드).
  4. 검은 창에서 app.py 폴더로 가서: git init 쳐.
  5. git add . 쳐서 파일 추가.
  6. git commit -m "첫 코드" 쳐서 저장.
  7. GitHub에서 리포지토리 주소 복사 (https://github.com/너의이름/리포이름.git).
  8. 검은 창에 git remote add origin 그주소 쳐.
  9. git push -u origin main 쳐서 인터넷에 올리기.

이제 GitHub에 코드가 저장됐어! 친구들도 볼 수 있어.

Flask로 웹사이트 더 멋지게 만들기

기본 웹을 만들었으니, HTML 파일 추가해서 예쁘게 해. "templates" 폴더 만들고 "index.html" 파일 만들어:

<!DOCTYPE html> <html> <head><title>내 웹</title></head> <body><h1>환영해!</h1></body> </html>

app.py 코드 바꾸기: return render_template('index.html') 로.

다시 테스트 해보자!

API로 뉴스나 주식 정보 가져오기 (재미있는 데이터 넣기)

API는 인터넷에서 정보 가져오는 마법이야. 뉴스 API 쓰려면 키 받아야 해 (newsapi.org 가서 무료 키 받기).

  1. pip install requests 설치.
  2. app.py에 코드 추가: import requests
  3. 함수 안에: url = 'https://newsapi.org/v2/top-headlines?country=kr&apiKey=너의키'
  4. response = requests.get(url)
  5. data = response.json()
  6. 웹에 data 보여주기.

주식도 비슷해 (alphavantage.co 키 받기). 천천히 따라 해!

HTML 코드 공부하기 (웹 페이지 꾸미기)

HTML은 웹 페이지 만드는 기본이야. 아래 제목 클릭해서 공부해!

  • 제목 만들기
  • 문단 만들기
  • 목록 만들기
  • 그림 넣기

제목 만들기

제목은 <h1>부터 <h6>까지. 큰 제목은 h1이야.

따라 하기: 파일에 <h1>큰 제목</h1> 써서 저장. 브라우저 열어 봐!

<h1>안녕!</h1>

문단 만들기

문단은 <p>로. 긴 글 쓰기 좋아.

따라 하기: <p>이건 문단이야!</p> 써봐.

<p>재미있는 이야기</p>

목록 만들기

목록은 <ul>로 (점 목록) 또는 <ol>로 (숫자 목록).

따라 하기: <ul><li>사과</li><li>바나나</li></ul>

<ul> <li>1번</li> <li>2번</li> </ul>

그림 넣기

그림은 <img src="그림주소">로.

따라 하기: 인터넷 그림 주소 복사해서 써봐.

<img src="https://example.com/image.jpg" alt="그림">

인터넷에 웹사이트 올리기 (배포하기)

GitHub에 저장했으니, 이제 전 세계에 보여주자. PythonAnywhere 써보자 (무료!)

  1. pythonanywhere.com 가서 계정 만들기.
  2. 로그인 후 "Web" 탭 클릭, "Add a new web app" 누르기.
  3. Flask 선택하고, GitHub 리포지토리 연결 (GitHub 계정 연동).
  4. 리포지토리 선택하고 설치. "Reload" 누르면 웹 주소 나와!
  5. 주소 클릭해서 봐. 친구들에게 공유해!

다른 방법: Render.com 가서 GitHub 연결하고 배포. 비슷해!

Flask를 어디에 쓸 수 있어? (유용한 범위)

Flask는 블로그, 게임 사이트, 뉴스 앱, 주식 정보 사이트 등 만들 수 있어. 학교 프로젝트나 재미로도 좋아. API로 데이터 가져와서 더 똑똑하게!
