Flask와 Python 기본 개념

Python이란?

Python은 1991년에 발표된 고급 프로그래밍 언어로, 가독성이 높고 간결한 문법이 특징입니다. 데이터 분석, 인공지능, 웹 개발 등 다양한 분야에서 널리 사용됩니다.

초보자부터 전문가까지 모두가 사용할 수 있으며, 방대한 라이브러리 생태계를 자랑합니다.

Flask란?

Flask는 Python으로 작성된 마이크로 웹 프레임워크입니다. '마이크로'라는 이름은 핵심 기능만을 제공함으로써 가볍고 유연한 개발 환경을 제공한다는 의미입니다.

Flask는 작은 프로젝트부터 복잡한 웹 애플리케이션까지 다양한 규모의 개발에 적합합니다.

Flask와 Python의 관계

Flask는 Python으로 작성되었으며, Python의 강력한 기능을 웹 개발에 활용할 수 있게 해줍니다.

Python으로 비즈니스 로직을 작성하고, Flask는 이를 웹 인터페이스로 노출시키는 역할을 합니다. 이 조합은 빠른 프로토타이핑과 유연한 개발을 가능하게 합니다.

Flask의 핵심 개념

  • 라우팅(Routing): URL을 Python 함수에 매핑하는 기능
  • 템플릿(Templates): HTML을 동적으로 생성하는 시스템
  • 요청(Request) 처리: 사용자 입력 및 데이터 처리
  • 응답(Response) 생성: 웹 브라우저에 반환할 결과 생성
  • 미들웨어(Middleware): 요청 처리 전후에 실행되는 기능

Flask와 Python의 장단점

장점

  • 간결하고 가벼움: 작은 코드로도 웹 서버 구축 가능
  • 유연한 확장성: 필요한 기능만 선택해 추가 가능
  • 파이썬 생태계 활용: 데이터 분석, 머신러닝 등 다양한 라이브러리 통합
  • 빠른 개발 속도: 프로토타입 제작과 테스트에 적합
  • 쉬운 학습 곡선: 초보자도 쉽게 배울 수 있음
  • 풍부한 문서화: 공식 문서와 커뮤니티 자료가 풍부

단점

  • 대규모 프로젝트 제한: Django에 비해 확장성이 낮음
  • 기본 기능 부족: 많은 기능을 직접 구현하거나 확장 필요
  • 비동기 처리: 기본적으로 비동기 지원이 약함
  • 보안 기능: 기본 보안 기능이 Django보다 부족
  • 성능 한계: 매우 높은 트래픽에는 적합하지 않을 수 있음

Flask vs Django

Flask와 Django는 Python의 대표적인 웹 프레임워크입니다. Django는 "배터리 포함" 철학으로 모든 기능을 제공하는 반면, Flask는 최소한의 기능을 제공하고 개발자가 필요한 부분만 선택해 사용합니다.

Flask는 작은 프로젝트나 마이크로서비스에, Django는 대규모 엔터프라이즈 애플리케이션에 적합합니다.

Flask와 Python의 사용 용도

웹 API 서버

Flask는 RESTful API 구축에 매우 적합합니다. 간단한 설정으로 JSON 기반의 API를 빠르게 개발할 수 있습니다.

데이터베이스와 연동하여 CRUD(Create, Read, Update, Delete) 기능을 구현하기 용이합니다.

데이터 대시보드

Python의 데이터 분석 라이브러리(pandas, matplotlib)와 Flask를 결합하면 실시간 데이터 시각화 대시보드를 만들 수 있습니다.

주식 분석, 비즈니스 인텔리전스, IoT 데이터 모니터링 등에 활용됩니다.

마이크로서비스

경량인 Flask는 마이크로서비스 아키텍처에 적합합니다. 각 서비스를 독립적으로 개발하고 배포할 수 있습니다.

서비스 간 통신은 REST API나 메시지 큐를 통해 이루어집니다.

Flask의 실제 활용 사례

  • Netflix: 콘텐츠 전송 네트워크 관리 도구
  • Airbnb: 내부 도구 및 마이크로서비스
  • Uber: 배달 예측 및 분석 도구
  • Lyft: 지리공간 데이터 처리
  • Reddit: 초기 버전 개발

왕초보 Flask 따라하기

1

설치하기

Python이 설치되어 있다면, 터미널에서 다음 명령어로 Flask를 설치합니다:

# Flask 설치 명령어
pip install flask
2

기본 앱 생성

app.py 파일을 생성하고 다음 코드를 입력합니다:

from flask import Flask

app = Flask(__name__)

# 홈페이지 라우트
@app.route('/')
def home():
    return "안녕하세요! Flask 첫 페이지입니다!"

if __name__ == '__main__':
    app.run(debug=True)
3

애플리케이션 실행

터미널에서 다음 명령어로 애플리케이션을 실행합니다:

# 애플리케이션 실행
python app.py

브라우저에서 http://localhost:5000 주소로 접속하면 결과를 확인할 수 있습니다.

4

라우트 추가하기

새로운 페이지를 추가하려면 다음과 같이 라우트를 추가합니다:

# 정보 페이지 추가
@app.route('/info')
def info():
    return "이것은 정보 페이지입니다."

Flask에서 HTML 템플릿 사용하기

1

폴더 구조 생성

프로젝트에 다음과 같은 폴더 구조를 생성합니다:

project/
├── app.py
└── templates/
    └── index.html

templates 폴더가 Flask에서 HTML 템플릿을 찾는 기본 위치입니다.

2

HTML 파일 생성

templates/index.html 파일을 생성하고 기본 HTML 구조를 작성합니다:

<!DOCTYPE html>
<html>
<head>
  <title>나의 첫 Flask 페이지</title>
</head>
<body>
  <h1>환영합니다!</h1>
  <p>이 페이지는 Flask로 만들었어요.</p>
</body>
</html>
3

Flask에서 HTML 렌더링

app.py를 수정하여 HTML 템플릿을 렌더링합니다:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')
4

동적 데이터 전달

Python에서 HTML로 데이터를 전달할 수 있습니다:

@app.route('/')
def home():
    message = "Flask에서 보낸 메시지"
    return render_template('index.html', message=message)

HTML에서는 다음과 같이 데이터를 사용합니다:

<body>
  <h1>{{ message }}</h1>
</body>

다중 페이지 웹사이트 만들기

1

라우트 추가

여러 페이지를 위한 라우트를 추가합니다:

# 메인 페이지
@app.route('/')
def home():
    return render_template('index.html')

# Flask 소개 페이지
@app.route('/flask')
def flask_intro():
    return render_template('flask.html')

# Python 소개 페이지
@app.route('/python')
def python_intro():
    return render_template('python.html')
2

템플릿 상속

공통 레이아웃을 위한 base.html 생성:

<!DOCTYPE html>
<html>
<head>
  <title>{% block title %}{% endblock %}</title>
</head>
<body>
  <nav>
    <a href="/">홈</a>
    <a href="/flask">Flask</a>
    <a href="/python">Python</a>
  </nav>

  {% block content %}{% endblock %}

<footer>© 2023 내 웹사이트</footer>
</body>
</html>
3

페이지별 템플릿 생성

flask.html 예시:

{% extends "base.html" %}

{% block title %}Flask 소개{% endblock %}

{% block content %}
  <h1>Flask란?</h1>
  <p>Flask는 Python 웹 프레임워크입니다.</p>
{% endblock %}
4

동적 URL 파라미터

URL에서 파라미터를 받아 처리:

@app.route('/user/<username>')
def show_user(username):
    return render_template('user.html', name=username)

user.html에서 {{ name }}으로 사용자 이름을 표시할 수 있습니다.

Flask에서 주식 API 구현

Flask를 사용하여 주식 데이터를 가져오는 API를 구현하는 방법:

1

필요 라이브러리 설치

pip install flask yfinance pandas
2

주식 데이터 엔드포인트 생성

import yfinance as yf
from flask import jsonify

@app.route('/stock/<ticker>')
def get_stock_data(ticker):
    stock = yf.Ticker(ticker)
    # 주식 정보 가져오기
    info = stock.info
    # 주가 기록 가져오기
    history = stock.history(period="1y")
    # JSON 형식으로 반환
    return jsonify({
        'info': info,
        'history': history.reset_index().to_dict('records')
    })
3

프론트엔드에서 데이터 가져오기

JavaScript를 사용하여 Flask API에서 데이터를 가져옵니다:

// 주식 데이터 가져오기
fetch(`/stock/${ticker}`)
  .then(response => response.json())
  .then(data => {
    // 데이터 처리 및 차트 생성
    renderStockChart(data.history);
    displayStockInfo(data.info);
  });
