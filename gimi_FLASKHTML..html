파이썬 & 플라스크 웹 개발 입문

왕초보를 위한 가장 쉬운 웹 개발 가이드

1. 파이썬(Python)이란 무엇일까요?

+

파이썬은 컴퓨터와 대화하기 위한 매우 쉬운 언어라고 생각하면 됩니다. 마치 우리가 사람과 한국어로 대화하듯, 컴퓨터에게 파이썬이라는 언어로 명령을 내리는 것이죠.

레고 블록에 비유해볼까요? 파이썬은 다양한 모양의 레고 블록과 같습니다. 이 블록들을 조립해서 자동차, 집, 로봇 등 원하는 것은 무엇이든 만들 수 있죠. 파이썬으로도 간단한 계산기부터 복잡한 인공지능까지 정말 다양한 프로그램을 만들 수 있습니다.

  • 문법이 쉬워요: 영어와 비슷해서 배우기 쉽고 코드가 간결합니다.
  • 활용 분야가 넓어요: 웹 개발, 데이터 분석, 인공지능, 게임 등 못하는 게 거의 없어요.
  • '배터리 포함': 유용한 기능들이 기본적으로 많이 포함되어 있어 바로 사용할 수 있습니다.

2. 플라스크(Flask)란 무엇일까요?

+

플라스크는 파이썬으로 웹사이트를 만들 수 있게 도와주는 특별한 도구 상자입니다.

다시 레고 비유를 들어볼게요. 파이썬이 레고 블록이라면, 플라스크는 '웹사이트 만들기 전용 레고 키트'와 같습니다. 이 키트 안에는 웹사이트의 뼈대를 만들고, 인터넷 주소(URL)에 따라 다른 페이지를 보여주는 등 웹 개발에 꼭 필요한 부품들이 들어있습니다.

  • 가볍고 유연해요: 꼭 필요한 핵심 기능만 있어서 시작하기 쉽습니다. 필요한 기능은 레고 블록처럼 하나씩 추가하면 됩니다.
  • '마이크로' 프레임워크: '마이크로'라는 말은 기능이 없다는 뜻이 아니라, 여러분이 원하는 대로 자유롭게 만들 수 있다는 의미입니다.
  • 확장성이 좋아요: 작은 웹사이트부터 큰 서비스까지 모두 만들 수 있습니다.

3. 파이썬과 플라스크는 어떤 관계인가요?

+

아주 간단합니다. 플라스크는 파이썬으로 만들어진 도구입니다. 따라서 플라스크를 사용하려면 반드시 파이썬이 설치되어 있어야 합니다.

우리는 파이썬 언어를 사용해서 플라스크라는 도구에게 "이 주소로 접속하면 이 페이지를 보여줘" 라고 명령을 내리는 것입니다.

파이썬 (언어) ➡️ 플라스크 (도구) ➡️ 웹사이트 (결과물)

4. 파이썬과 플라스크 설치하고 웹 만들어보기

+

1단계: 파이썬 설치하기

먼저 컴퓨터에 파이썬을 설치해야 합니다. 공식 홈페이지에서 최신 버전을 다운로드하세요. 설치할 때 "Add Python to PATH" 체크박스를 꼭! 클릭해주세요.

2단계: 플라스크 설치하기

Windows에서는 '명령 프롬프트(cmd)'나 'PowerShell'을, Mac에서는 '터미널'을 열고 아래 명령어를 입력하세요. pip는 파이썬의 앱스토어 같은 역할을 합니다. 

pip install Flask

3단계: 세상에서 가장 간단한 웹 앱 만들기

원하는 곳에 폴더를 하나 만들고, 그 안에 app.py 라는 이름으로 파일을 만드세요. 그리고 아래 코드를 그대로 복사해서 붙여넣으세요. 

# 1. 플라스크 도구를 가져옵니다.
from flask import Flask

# 2. 플라스크를 이용해 웹 앱을 만듭니다.
app = Flask(__name__)

# 3. 특정 주소(URL)로 접속하면 실행될 함수를 지정합니다.
@app.route('/')
def hello_world():
    return 'Hello, World! 제 첫 웹사이트에 오신 것을 환영합니다!'

# 4. 이 파이썬 파일을 직접 실행하면 웹 서버를 가동합니다.
if __name__ == '__main__':
    app.run(debug=True)

4단계: 웹 서버 실행하기

터미널에서 app.py 파일이 있는 폴더로 이동한 후, 아래 명령어를 실행하세요. 

python app.py

터미널에 Running on http://127.0.0.1:5000/ 와 같은 메시지가 나타나면 성공입니다! 이제 웹 브라우저를 열고 주소창에 http://127.0.0.1:5000 을 입력해보세요. 화면에 "Hello, World!..." 메시지가 보일 겁니다.

5. 기본 HTML 만들고 웹에 표시하기

+

단순한 글자 대신, 진짜 웹페이지처럼 꾸미려면 HTML 파일이 필요합니다.

1단계: `templates` 폴더 만들기

app.py 파일이 있는 곳에 templates 라는 이름의 폴더를 만드세요. 플라스크는 이 폴더 안에서 HTML 파일을 찾습니다.

2단계: `index.html` 파일 만들기

방금 만든 templates 폴더 안에 index.html 파일을 만들고 아래 내용을 붙여넣으세요. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>나의 첫 웹페이지</title>
</head>
<body>
    <h1>플라스크와 HTML의 만남!</h1>
    <p>파이썬 코드가 이 HTML 파일을 불러왔어요.</p>
</body>
</html>

3단계: `app.py` 수정하기

이제 파이썬 코드가 HTML 파일을 불러오도록 app.py를 수정합시다. 

# render_template 라는 도구를 추가로 가져옵니다. HTML 파일을 불러오는 역할을 합니다.
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    # 'Hello World' 대신 index.html 파일을 화면에 보여줍니다.
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

서버를 껐다가(Ctrl+C) 다시 켜고(python app.py) 브라우저를 새로고침하면, 방금 만든 HTML 페이지가 나타날 겁니다!

6. API를 이용해 최신 뉴스 보여주기

+

API는 '데이터를 배달해주는 웨이터' 같은 것입니다. 우리는 API를 통해 다른 서비스(뉴스, 주식, 날씨 등)의 데이터를 우리 웹사이트에 가져와 보여줄 수 있습니다.

1단계: `requests` 라이브러리 설치

API에 데이터를 요청하려면 `requests` 라는 도구가 필요합니다. 터미널에 아래 명령어를 입력해 설치하세요. 

pip install requests

2단계: `app.py`에서 뉴스 데이터 가져오기

여기서는 실제 API 키가 필요 없는 간단한 예제용 뉴스 데이터를 사용하겠습니다. app.py를 아래와 같이 수정하세요. 

from flask import Flask, render_template
import requests # requests 라이브러리 가져오기

app = Flask(__name__)

@app.route('/news')
def get_news():
    # 예제: IT 분야 최신 뉴스 API (실제 API는 주소가 다를 수 있습니다)
    # 여기서는 간단한 실습을 위해 미리 준비된 JSON 데이터를 사용합니다.
    dummy_news_data = {
        "articles": [
            {"title": "파이썬, 2025년 가장 인기있는 언어로 선정", "content": "파이썬의 쉬운 문법과 강력한 라이브러리가 인기 비결입니다."},
            {"title": "플라스크 3.0 출시, 비동기 지원 강화", "content": "더 빠르고 효율적인 웹 개발이 가능해졌습니다."},
            {"title": "API 경제, 개발자에게 새로운 기회 제공", "content": "다양한 서비스의 데이터를 조합하여 혁신적인 앱을 만들 수 있습니다."}
        ]
    }
    articles = dummy_news_data["articles"]
    
    # news.html 파일에 'articles'라는 이름으로 뉴스 데이터를 전달합니다.
    return render_template('news.html', articles=articles)

if __name__ == '__main__':
    app.run(debug=True)

3단계: `news.html` 만들기

templates 폴더에 news.html 파일을 만들고, 파이썬에서 받은 뉴스 데이터를 표시하도록 코드를 작성합니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>최신 IT 뉴스</title>
    <style>
        body { font-family: sans-serif; margin: 2em; }
        .article { border: 1px solid #ccc; padding: 1em; margin-bottom: 1em; border-radius: 5px; }
        h2 { color: #2c3e50; }
    </style>
</head>
<body>
    <h1>오늘의 최신 IT 뉴스</h1>
    
    <!-- 파이썬에서 받은 articles 데이터를 반복해서 보여줍니다. -->
    {% for article in articles %}
    <div class="article">
        <h2>{{ article.title }}</h2>
        <p>{{ article.content }}</p>
    </div>
    {% endfor %}
    
</body>
</html>

서버를 재시작하고, 웹 브라우저에서 http://127.0.0.1:5000/news 로 접속해보세요. 우리가 준비한 가짜 뉴스 목록이 멋지게 나타날 겁니다! 주식 데이터도 이와 동일한 원리로 가져와 분석하고 표시할 수 있습니다.

7. 내 웹사이트를 인터넷에 올리는 법 (배포)

+

지금까지 만든 웹사이트는 내 컴퓨터에서만 접속할 수 있습니다. 친구들에게 자랑하려면 인터넷 세상에 공개해야겠죠? 이 과정을 '배포'라고 합니다. 초보자가 가장 쉽게 배포할 수 있는 방법 몇 가지를 소개합니다.

  • PythonAnywhere: 파이썬 웹 앱을 위한 가장 쉬운 호스팅 서비스 중 하나입니다. 무료 플랜으로 간단한 플라스크 앱을 몇 번의 클릭만으로 배포할 수 있습니다.
  • Render: 최근 인기를 얻고 있는 클라우드 서비스로, 무료 플랜을 제공하며 GitHub 계정과 연동하여 코드를 올리면 자동으로 배포해주는 기능이 매우 편리합니다.
  • Vercel / Netlify (정적 사이트용): 만약 파이썬 백엔드 없이 순수 HTML, CSS, JavaScript로만 만든 '정적 사이트'라면 이 서비스들이 최고입니다. 파일을 드래그 앤 드롭하는 것만으로 배포가 끝납니다.

이러한 서비스들은 대부분 자세한 가이드를 제공하므로, "Render Flask deploy" 와 같이 검색하여 차근차근 따라 해보는 것을 추천합니다.
