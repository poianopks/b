이 자료는 프로그래밍 언어 파이썬(Python)의 가장 기초적인 개념 30가지를 유치원생, 초등학생, 중학생의 눈높이에 맞춰 설명하는 학습 가이드입니다. 각 연령대에 맞춰 비유, 놀이, 간단한 예시, 코드 조각 등을 활용하여 컴퓨터 과학의 기본 원리와 파이썬의 기초 문법을 쉽고 재미있게 배울 수 있도록 구성했습니다. 순서대로 따라 하며 컴퓨팅 사고력과 파이썬 기초 실력을 키워보세요.

1. 명령어 (Command)

컴퓨터에게 무엇을 해야 할지 알려주는 구체적인 지시입니다.

유치원 수준 (놀이와 생각): "이렇게 해!" 라고 말하기

비유/이야기: 엄마나 선생님이 "장난감 정리하자!", "손 씻으세요!" 라고 말하는 것처럼, 컴퓨터에게도 우리가 무엇을 원하는지 하나하나 알려줘야 해요. 이 하나하나의 지시를 '명령어'라고 불러요. 로봇 친구에게 "앞으로 한 칸 가!", "오른쪽으로 돌아!" 라고 말하는 것과 같아요.

활동: '선생님 따라 하기' 놀이: 선생님이 "점프!", "박수!" 같은 명령어를 말하면 그대로 따라 해 보세요. 우리가 선생님의 명령어를 듣고 행동하는 것처럼, 컴퓨터도 우리의 명령어를 듣고 움직여요.

초등학생 수준 (탐구와 구축): 컴퓨터에게 내리는 지시

설명: 명령어는 컴퓨터가 알아들을 수 있는 언어로 내리는 구체적인 지시 사항이에요. 예를 들어 그림을 그리게 하거나, 글자를 화면에 보여주게 하거나, 계산을 시킬 때 사용해요. [span_0](start_span)[span_0](end_span) 파이썬에서는 print() 같은 단어가 명령어 역할을 해요.

중학생 수준 (코드와 창작): 실행 가능한 코드 단위

정의: 명령어(Command 또는 Statement)는 프로그래밍 언어에서 컴퓨터가 특정 작업을 수행하도록 지시하는 가장 기본적인 단위입니다. 파이썬 인터프리터가 해석하고 실행할 수 있는 코드 한 줄 또는 여러 줄이 될 수 있습니다. [span_1](start_span)[span_1](end_span)

# 각 줄이 하나의 명령어입니다.
print("안녕하세요!") # 화면에 "안녕하세요!"를 출력하는 명령어
user_name = "파이썬" # "파이썬"이라는 값을 user_name 변수에 저장하는 명령어
level = level + 1    # level 변수의 값을 1 증가시키는 명령어
2. 순서 (Sequence)

명령어들이 실행되는 정해진 차례입니다.

유치원 수준 (놀이와 생각): 차례차례 따라 하기

비유/이야기: 아침에 일어나서 하는 일들을 생각해 볼까요? '일어나기 → 세수하기 → 옷 입기 → 아침 먹기' 처럼 순서대로 하죠? 컴퓨터에게 일을 시킬 때도 어떤 일을 먼저 하고, 어떤 일을 나중에 할지 순서를 정해주는 것이 중요해요. 이것이 바로 '순서'예요. 3. 파이썬이란? (What is Python?)

사람들이 컴퓨터 프로그램을 만들기 위해 사용하는 인기 있는 프로그래밍 언어 중 하나입니다.

유치원 수준 (놀이와 생각): 컴퓨터랑 이야기하는 특별한 말

비유/이야기: 우리가 친구들과 이야기할 때 한국말을 쓰는 것처럼, 컴퓨터랑 이야기할 때는 특별한 말이 필요해요. '파이썬'은 컴퓨터가 알아들을 수 있는 특별한 말(언어) 중 하나예요. 이 말로 컴퓨터에게 그림을 그리게 하거나 재미있는 소리를 내게 할 수 있어요.

초등학생 수준 (탐구와 구축): 무언가를 만드는 도구

설명: 파이썬은 사람들이 컴퓨터 프로그램, 게임, 웹사이트, 로봇 등을 만드는 데 사용하는 인기 있는 도구(프로그래밍 언어)예요. [span_14](start_span)[span_14](end_span)[span_16](start_span)[span_16](end_span) 영어를 닮아서 배우기 쉽고 [span_18](start_span)[span_18](end_span)[span_19](start_span)[span_19](end_span)[span_20](start_span)[span_20](end_span), 많은 사람들이 사용하고 있어요.

중학생 수준 (코드와 창작): 고수준 인터프리터 언어

정의: 파이썬은 귀도 반 로섬(Guido van Rossum)이 개발한 고수준(high-level), 인터프리터(interpreted), 범용(general-purpose) 프로그래밍 언어입니다. [span_15](start_span)[span_15](end_span)[span_17](start_span)[span_17](end_span) '고수준'은 사람이 이해하기 쉬운 문법을 가졌다는 의미이고, '인터프리터'는 코드를 한 줄씩 바로 실행한다는 의미입니다.

4. 데이터/정보 (Data/Information)

컴퓨터가 다루는 숫자, 글자, 참/거짓과 같은 정보의 종류입니다.

유치원 수준 (놀이와 생각): 여러 가지 종류의 물건들

비유/이야기: 장난감 상자에 여러 종류의 장난감(블록, 인형, 자동차)이 있거나, 접시에 여러 종류의 음식(과일, 채소, 과자)이 있는 것처럼, 컴퓨터도 여러 종류의 정보를 다뤄요. 숫자 정보, 글자 정보, '맞다/틀리다' 같은 생각 정보가 있어요. (컴퓨팅 사고력: 데이터 표현)

초등학생 수준 (탐구와 구축): 숫자, 글자, 그리고 참/거짓

설명: 컴퓨터는 여러 종류의 정보, 즉 '데이터 타입(data type)'을 저장하고 사용해요. 가장 기본적인 종류는 숫자(나이, 점수 등), 글자/단어(이름, 메시지 등, '문자열'이라고 불러요), 그리고 '참' 또는 '거짓'으로 답하는 것('불리언'이라고 불러요, 예를 들어 게임이 끝났는지 아닌지)이에요. 중학생 수준 (코드와 창작): 데이터 타입 (int, float, str, bool)

정의: 데이터 타입은 변수에 저장될 데이터의 종류를 명시하며, 해당 데이터로 수행할 수 있는 연산을 결정합니다. 파이썬의 기본 데이터 타입으로는 정수(int), 실수(float), 문자열(str), 불리언(bool) 등이 있습니다. [span_27](start_span)[span_27](end_span)[span_28](start_span)[span_28](end_span)[span_29](start_span)[span_29](end_span)

5. 변수 (Variables)

데이터(정보)를 저장하는 상자에 붙인 이름표입니다.

유치원 수준 (놀이와 생각): 이름표 붙은 상자

중학생 수준 (코드와 창작): 데이터 저장 및 참조

정의: 변수(Variable)는 데이터를 저장하기 위해 컴퓨터 메모리 공간에 할당된 이름입니다. 변수에 저장된 값은 프로그램 실행 중에 변경될 수 있습니다. 의미 있는 변수 이름을 사용하는 것이 코드 가독성에 중요합니다. [span_40](start_span)[span_40](end_span)

6. 출력 (Printing / Output)

컴퓨터가 우리에게 결과를 보여주는 방법입니다.

7. 주석 (Comments)

코드에 대한 설명을 적어두는 메모로, 컴퓨터는 무시합니다.

유치원 수준 (놀이와 생각): 그림 뒤에 숨겨둔 비밀 메모

비유/이야기: 내가 그린 그림 뒷면에 나만 보거나 엄마 아빠에게만 보여줄 비밀 메모를 적는다고 생각해 보세요. 코드 속의 '주석'도 이와 같이 컴퓨터는 읽지 않고 사람만 읽는 메모예요.

초등학생 수준 (탐구와 구축): 코드 설명하기

설명: 주석은 코드 안에 써넣는 설명글이에요. 컴퓨터는 주석을 무시하고 그냥 지나가요. 주석은 사람이 코드를 읽을 때 '이 코드는 이런 일을 하는구나' 하고 이해하기 쉽게 도와주는 역할을 해요. [span_45](start_span)[span_45](end_span)[span_46](start_span)[span_46](end_span)[span_47](start_span)[span_47](end_span)

정의: 주석(Comment)은 소스 코드 내에 포함된 설명문으로, 코드의 가독성을 높이고, 복잡한 로직을 설명하거나, 특정 코드 부분을 임시로 비활성화(주석 처리)하는 등의 목적으로 사용됩니다. 인터프리터는 주석 부분을 실행하지 않습니다.

8. 기본 수학 연산자 (+, -, *, /)

더하기, 빼기, 곱하기, 나누기 같은 기본적인 계산을 하는 기호입니다.

초등학생 수준 (탐구와 구축): 컴퓨터 계산기

설명: 파이썬은 계산기처럼 수학 계산을 할 수 있어요. 더하기는 +, 빼기는 -, 곱하기는 *, 나누기는 / 기호(연산자라고 불러요)를 사용해요. [span_48](start_span)[span_48](end_span)[span_49](start_span)[span_49](end_span)[span_50](start_span)[span_50](end_span)[span_51](start_span)[span_51](end_span)

중학생 수준 (코드와 창작): 산술 연산자

정의: 산술 연산자(Arithmetic Operator)는 숫자 데이터에 대해 기본적인 수학 연산을 수행하는 기호입니다. 추가적으로 //(정수 나눗셈 - 몫), %(나머지 연산), **(거듭제곱) 등도 자주 사용됩니다. [span_52](start_span)[span_52](end_span)[span_54](start_span)[span_54](end_span)

print(2 ** 3)  # 8 (2의 3제곱)
9. 문자열 (Strings)
