주식 종목 분석, 뉴스 확인하는 법

할아버지도 이해하는 쉬운 방법으로 주식 분석 기초를 배웁니다.

주식이 뭔지 이해하기

주식은 회사의 작은 조각이라고 생각하세요. 예) 맛집 100조각 중 1조각을 사면 1% 주인이 됩니다.

정보는 어디서?

  • 네이버 금융 – 가장 쉽게 시작
  • 증권사 앱 – 실제 거래 및 공시
  • 인베스팅닷컴 – 심화 차트
2

파이썬으로 주식 데이터 가져오기

실전 예제 – 삼성전자 1년치 주가

import yfinance as yf import matplotlib.pyplot as plt samsung = yf.Ticker("005930.KS") data = samsung.history(period="1y") plt.plot(data.index, data["Close"]) plt.title("삼성전자 1년 주가") plt.savefig("img/samsung-chart.png")

위 코드를 Google Colab에 복사 → ▶ 실행만 하면 차트가 저장됩니다.
