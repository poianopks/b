30개 주제로 배우는 파이썬 프로그래밍 & Acode 앱 활용법

이 가이드는 파이썬 프로그래밍을 처음 배우는 분들을 위해 제작되었습니다. 30개의 핵심 주제로 파이썬 기초를 체계적으로 학습하고, 모바일 환경에서 파이썬을 작성하고 실행할 수 있는 Acode 앱의 사용법까지 모두 배울 수 있습니다.

각 주제를 클릭하면 상세 설명과 예제 코드를 볼 수 있습니다. 직접 코드를 작성해보며 파이썬 기초를 탄탄히 다져보세요!

파이썬 기초 30개 주제

파이썬 프로그래밍의 핵심 개념을 30개의 주제로 나누어 학습합니다. 각 주제를 클릭하면 상세 설명과 예제 코드를 확인할 수 있습니다.

1. 변수(Variables)

데이터를 저장하는 공간. 파이썬에서는 변수를 선언할 때 데이터 타입을 명시하지 않습니다.

name = "김파이"  # 문자열 변수
age = 10        # 정수형 변수
height = 150.5  # 실수형 변수
is_student = True  # 불리언 변수

2. 데이터 타입(Data Types)

파이썬의 기본 데이터 타입: 정수(int), 실수(float), 문자열(str), 불리언(bool)

# 데이터 타입 확인
print(type(10))       # <class 'int'>
print(type(3.14))     # <class 'float'>
print(type("Hello"))  # <class 'str'>
print(type(True))     # <class 'bool'>

3. 연산자(Operators)

산술, 비교, 논리 연산자 등 파이썬의 다양한 연산자 사용법

# 산술 연산자
result = 10 + 3 * 2  # 16

# 비교 연산자
is_adult = age >= 20  # True or False

# 논리 연산자
can_drive = has_license and age >= 18

4. 조건문(if문)

조건에 따라 다른 코드를 실행하는 분기문

score = 85

if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
else:
    grade = 'C'

5. 반복문(for문)

특정 횟수만큼 또는 컬렉션의 요소들을 순회하며 반복 실행

# 숫자 범위 반복
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# 리스트 순회
fruits = ['사과', '바나나', '딸기']
for fruit in fruits:
    print(fruit)

6. 반복문(while문)

조건이 참인 동안 코드 블록을 반복 실행

count = 0
while count < 5:
    print(count)
    count += 1  # 0, 1, 2, 3, 4 출력

7. 리스트(Lists)

순서가 있는 변경 가능한(mutable) 데이터 컬렉션

# 리스트 생성 및 조작
numbers = [1, 2, 3, 4, 5]
numbers.append(6)     # [1,2,3,4,5,6]
numbers.pop()         # 마지막 요소 제거
print(numbers[0])     # 첫번째 요소 출력

8. 튜플(Tuples)

순서가 있는 변경 불가능한(immutable) 데이터 컬렉션

# 튜플 생성
point = (10, 20)

# 튜플 요소 접근
x = point[0]  # 10
y = point[1]  # 20

9. 딕셔너리(Dictionaries)

키-값 쌍으로 이루어진 데이터 컬렉션

# 딕셔너리 생성
person = {
    'name': '김파이',
    'age': 25,
    'city': '서울'
}

# 값 접근
print(person['name'])  # 김파이

10. 집합(Sets)

중복되지 않는 고유한 요소들의 모음

# 집합 생성
fruits = {'사과', '바나나', '딸기', '사과'}
print(fruits)  # {'사과', '바나나', '딸기'} (중복 제거)

# 집합 연산
set1 = {1, 2, 3}
set2 = {3, 4, 5}
print(set1 | set2)  # 합집합: {1, 2, 3, 4, 5}

Acode 앱 사용법 가이드

Acode는 모바일에서 파이썬 코드를 작성하고 실행할 수 있는 강력한 코드 에디터 앱입니다. 아래 가이드를 따라 설치부터 사용법까지 배워보세요.

1. Acode 설치하기

구글 플레이 스토어에서 "Acode"를 검색하여 앱을 설치합니다. 설치 후 앱을 실행합니다.

2. 파이썬 플러그인 설치

Acode에서 파이썬을 실행하기 위해서는 플러그인이 필요합니다:

  1. 앱 우측 상단의 ≡ 메뉴 버튼을 탭합니다
  2. "플러그인" 섹션을 선택합니다
  3. "Python" 플러그인을 찾아 설치합니다
  4. 터미널 플러그인도 함께 설치합니다

3. 첫 파이썬 파일 만들기

새 파일을 생성하고 파이썬 코드를 작성합니다:

  1. 하단의 + 버튼을 눌러 새 파일 생성
  2. 파일 이름을 "hello.py"로 지정
  3. 에디터에 코드 작성: print("Hello Python!")

4. 코드 실행하기

작성한 파이썬 코드를 실행하는 방법:

  1. 상단의 플레이 버튼(▶)을 탭합니다
  2. 또는 ≡ 메뉴 > "실행"을 선택합니다
  3. 실행 결과는 하단의 콘솔에서 확인할 수 있습니다

5. HTML 파일 작성 및 미리보기

Acode에서 웹 개발도 가능합니다:

  1. 새 파일을 생성하고 이름을 "index.html"로 지정
  2. HTML 코드 작성 (아래 예제 참고)
  3. ≡ 메뉴 > "미리보기"를 선택하면 웹페이지 확인 가능
<!DOCTYPE html>
<html>
<head>
    <title>나의 첫 웹페이지</title>
</head>
<body>
    <h1>안녕하세요!</h1>
    <p>Acode로 만든 첫 웹페이지입니다.</p>
</body>
</html>
