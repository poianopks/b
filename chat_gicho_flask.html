🐍 Flask + HTML 왕초보 따라하기

1. Flask란 무엇인가요?

Python: 컴퓨터에게 말하는 언어예요. 아주 쉬운 문법으로 유명하죠!

Flask: Python으로 만든 작은 웹사이트를 쉽게 만들 수 있는 도구입니다.

→ 웹페이지(HTML)를 화면에 뿅! 보여주는 역할을 해요.

2. Flask 코드 예시

아래는 Flask를 이용해 HTML을 출력하는 코드입니다.

from flask import Flask, render_template_string

app = Flask(__name__)

@app.route('/')
def hello():
    return render_template_string("""
    <html>
        <head><title>Flask + HTML</title></head>
        <body style='text-align: center;'>
            <h1>안녕하세요!</h1>
            <p>Flask가 HTML을 보여주는 성공 예제입니다.</p>
        </body>
    </html>
    """)

if __name__ == '__main__':
    app.run(debug=True)

🔺 위 코드는 실제 Flask Python 예제 코드입니다.

3. 코드 설명 요약

1. flask에서 Flask, render_template_string 가져오기
2. Flask 웹 앱 만들기
3. '/' 주소에 접속하면 hello 함수 실행
4. HTML을 문자열로 직접 만들어 반환
5. 브라우저에서 결과 확인 가능 (http://127.0.0.1:5000)
