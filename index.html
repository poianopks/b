1. GPTs란 무엇인가? (중요 용어 설명)

GPTs(맞춤형 GPT)란?

GPTs는 OpenAI에서 제공하는 ‘맞춤형 AI 챗봇’입니다. 사용자가 직접 챗봇의 성격, 역할, 답변 방식, 외부 도구 연결 등을 설정해서 자신만의 AI 비서나 도우미를 만들 수 있습니다. 코딩 없이도 쉽게 만들 수 있어요[1][5][8].

GPT(Generative Pre-trained Transformer): OpenAI에서 개발한 대규모 언어 예측 모델로, 텍스트 생성, 요약, 번역 등 다양한 자연어 처리 작업을 수행합니다. GPT는 방대한 텍스트 데이터로 사전 학습되어, 문장의 흐름과 문맥을 이해하고 다음 단어를 예측하는 능력이 있습니다[1][10].

ChatGPT: GPT 모델을 활용한 대화형 AI 챗봇 서비스로, 사용자와 자연스럽게 대화하며 다양한 질문에 답변합니다[5][10].

GPTs: 사용자가 원하는 목적에 맞게 직접 챗봇을 커스터마이즈할 수 있는 맞춤형 GPT입니다. 예를 들어, 주식뉴스 요약, 종목 분석 봇 등 다양한 용도로 만들 수 있습니다[5][8][10].

2. API란 무엇인가?

API(에이피아이)란?

API는 ‘Application Programming Interface’의 약자로, 서로 다른 소프트웨어끼리 대화(데이터 주고받기)할 수 있도록 해주는 규칙입니다. 예를 들어, 네이버 지도에서 길찾기 정보를 가져오거나, 주식 시세를 받아오는 것처럼 외부 서비스의 기능을 내 앱이나 챗봇에서 사용할 수 있게 해줍니다[6].

3. API 스키마란 무엇인가?

API 스키마란?

API 스키마는 API가 어떻게 동작하는지, 어떤 데이터를 주고받는지 설명하는 ‘설명서’입니다. 마치 레고 조482조립 설명서처럼, API를 사용하는 방법과 구조를 쉽게 이해할 수 있게 해줍니다[3][6][7].

비유로 쉽게 설명하기

API 스키마는 ‘레고 설명서’와 같아요. 레고를 만들 때 설명서를 보고 조립하듯, API 스키마를 보고 챗봇이 외부 서비스와 데이터를 주고받는 방법을 알 수 있습니다. 개발자와 사용자가 서로 잘 이해할 수 있도록 ‘약속’을 정해두는 역할을 합니다[3][6][7].

4. API 키란 무엇인가?

API 키란?

API 키는 외부 서비스(예: 주식 시세, 뉴스 등)에 접근할 때 필요한 ‘비밀번호’입니다. 이 키가 있어야 내가 만든 챗봇이 외부 데이터를 가져올 수 있습니다.

5. Actions(액션) 기능이란?

Actions(액션) 기능이란?

Actions는 GPTs가 외부 서비스(API)와 연결되어, 실제로 데이터를 주고받을 수 있게 해주는 기능입니다. 예를 들어, 주식 시세를 실시간으로 가져오거나, 뉴스 요약을 외부 서비스에서 받아올 수 있습니다[2][6].

6. GPTs 만들기 따라하기

1단계: GPTs 만들기 시작

  1. ChatGPT에 로그인 후, 왼쪽 메뉴에서 "Explore" → "Create a GPT"를 클릭합니다.
  2. "Create"를 누르고, 원하는 챗봇 이름과 설명을 입력합니다.
    예시: 이름 "주식뉴스 분석 봇", 설명 "최신 주식뉴스와 종목 실시간 분석 제공"

2단계: 지시문(Instructions) 작성

예시 지시문:

사용자가 요청하면 최신 주식뉴스를 요약해서 알려주고, 종목명이나 키워드를 입력하면 관련 뉴스와 실시간 분석을 제공하세요. 분석은 API를 통해 최신 데이터를 받아와서 진행하세요.
자세히 따라하기 (클릭하면 이 섹션으로 이동)

7. API 연결(액션) 따라하기

1단계: Actions(액션) 추가

  1. Configure 탭에서 "Add Action"을 클릭합니다.
  2. API 스키마를 입력합니다.
    (예: 딥서치 뉴스 API의 OpenAPI 스펙을 복사해서 붙여넣기)
  3. 인증 정보에 API 키를 입력합니다.
  4. 테스트 버튼을 눌러 API가 잘 연결되는지 확인합니다.

8. API 스키마 입력 꼼꼼한 예제와 비유 설명

API 스키마 입력이란?

API 스키마는 외부 서비스(예: 뉴스, 주식 시세 등)와 어떻게 소통해야 하는지 알려주는 ‘설명서’입니다. GPTs의 Actions 기능에서 이 설명서를 입력하면, 챗봇이 외부 데이터를 가져올 수 있습니다[3][6][7].

비유로 쉽게 설명하기

API 스키마는 ‘레고 설명서’와 같아요. 레고를 만들 때 설명서를 보고 조립하듯, API 스키마를 보고 챗봇이 외부 서비스와 데이터를 주고받는 방법을 알 수 있습니다. 설명서가 없으면 어떻게 조립해야 할지 모르듯, 스키마가 없으면 챗봇도 외부 데이터를 어떻게 가져와야 할지 모릅니다[3][6][7].

예제: OpenAPI 스키마 기본 구조

openapi: 3.1.0
info:
  title: 주식뉴스 API
  description: 주식 관련 최신 뉴스를 제공하는 API
  version: 1.0.0
servers:
  - url: https://api.stocknews.com/v1
paths:
  /news:
    get:
      summary: 최신 주식뉴스 가져오기
      responses:
        '200':
          description: 성공
          content:
            application/json:
              schema:
                type: array
                items:
                  type: object
                  properties:
                    title:
                      type: string
                    url:
                      type: string
                    date:
                      type: string

이렇게 작성된 스키마를 GPTs의 Actions에 붙여넣으면, 챗봇이 주식뉴스를 가져올 수 있습니다.

9. 오픈API 무료로 구하는 방법

증권사/금융 OpenAPI 신청

  • 키움증권 OpenAPI+
    1. 키움증권 홈페이지에서 "OpenAPI+" 메뉴 클릭
    2. 회원가입/로그인 후, 사용 신청
    3. 공인인증서로 본인 인증
    4. 모의투자 신청(실제 거래 전 테스트 가능)
    5. API 키 발급 및 다운로드
  • 한국투자증권 OpenAPI
    1. 한국투자증권 개발자센터 접속
    2. 회원가입/로그인
    3. API 사용 신청(간단한 가입 절차)
    4. API 키 발급
  • 공공데이터포털 주식시세정보
    1. data.go.kr 접속
    2. "금융위원회 주식시세정보" 검색
    3. 활용신청(즉시 승인)
    4. 인증키 발급

뉴스 OpenAPI 신청

  • 딥서치 뉴스 API
    1. 딥서치 뉴스 API 사이트 접속
    2. 무료 테스트 신청(회원가입/로그인)
    3. API 키 발급
  • 네이버 뉴스 API
    1. 네이버 개발자센터 접속
    2. 애플리케이션 등록
    3. API 키 발급
