DAY 1

1일차 · 기본 설정 & 준비

🎯 오늘의 목표

마누스 AI를 사용하기 위한 기본 설정을 완료하고, 브라우저 오퍼레이터를 활성화합니다.

1-1. 마누스 계정 준비

1마누스 가입하기

1) 크롬 브라우저에서 manus.im에 접속합니다.

2) "Get started" 버튼을 클릭하고 구글 계정으로 가입합니다.

3) 가입이 완료되면 마누스 대시보드가 열립니다.

2브라우저 오퍼레이터 설치

4) 좌측 메뉴에서 "Connectors"를 클릭합니다.

5) "My Browser" 토글을 켭니다.

6) 크롬 웹 스토어에서 마누스 확장 프로그램을 설치합니다.

브라우저 오퍼레이터는 마누스의 핵심 기능입니다. 이것이 있어야 AI가 실제로 웹사이트에서 작업을 수행할 수 있습니다.

1-2. 필수 커넥터 연결

다음 서비스들을 연결하면 더 강력한 자동화가 가능합니다:

📧 Gmail

이메일 자동화, 알림 받기

📅 Google Calendar

일정 관리, 예약 작업

💻 GitHub

코드 관리, 웹사이트 배포

🌐 My Browser

브라우저 자동화 (필수!)

1-3. 첫 번째 테스트

설정이 완료되었는지 간단히 테스트해 봅니다:

테스트 프롬프트 
구글에서 "오늘 날씨"를 검색하고
검색 결과 첫 페이지의 내용을 요약해 줘.
마누스가 구글을 열고 검색을 수행한 후 결과를 요약해 주면 설정이 완료된 것입니다!
DAY 2

2일차 · AI 자동 쇼핑

🎯 오늘의 목표

마누스 AI가 쇼핑몰에서 상품을 검색하고, 비교하고, 장바구니에 담는 것까지 자동으로 하게 합니다.

2-1. 상품 검색 자동화

마누스에게 원하는 상품을 찾아달라고 요청합니다:

입력 예시 1 - 기본 검색 
쿠팡에 접속해서 "무선 블루투스 이어폰"을 검색해 줘.
검색 결과 중 평점 4.5 이상인 제품 5개를 찾아서
제품명, 가격, 평점, 리뷰 수를 표로 정리해 줘.
입력 예시 2 - 조건부 검색 
11번가에서 "겨울 패딩"을 검색해 줘.
조건:
- 가격: 10만원 ~ 20만원
- 평점: 4.0 이상
- 무료배송

조건에 맞는 상품 10개를 찾아서 정리해 줘.

2-2. 가격 비교 자동화

여러 쇼핑몰의 가격을 한 번에 비교할 수 있습니다:

입력 예시 
"애플 에어팟 프로 2세대" 제품을
쿠팡, 11번가, G마켓에서 각각 검색해서
가격을 비교해 줘.

각 쇼핑몰별로:
- 최저가
- 배송비
- 예상 도착일
을 정리해서 어디서 사는 게 가장 좋은지 추천해 줘.

2-3. 장바구니 담기 자동화

입력 예시 
쿠팡에서 방금 찾은 제품 중
가장 가성비가 좋은 제품을 골라서
장바구니에 담아 줘.
결제는 직접 하셔야 합니다! 마누스는 장바구니 담기까지만 자동화하고, 실제 결제는 사용자가 확인 후 진행합니다.

2-4. 쇼핑 리스트 자동 관리

입력 예시 
다음 쇼핑 리스트의 각 항목을 쿠팡에서 검색해서
최저가 제품을 찾아 줘:

1. 샴푸 1000ml
2. 칫솔 4개입
3. 휴지 30롤
4. 세제 3kg

각 항목별 추천 제품과 가격을 정리하고,
총 예상 금액도 계산해 줘.
DAY 3

3일차 · 인터넷 자료 검색 자동화

🎯 오늘의 목표

마누스 AI가 인터넷에서 자료를 검색하고, 정리하고, 요약하는 방법을 배웁니다.

3-1. 기본 자료 검색

입력 예시 
"2024년 한국 전기차 시장 동향"에 대해 조사해 줘.

다음 내용을 포함해서 정리해 줘:
1. 시장 규모와 성장률
2. 주요 브랜드별 점유율
3. 정부 지원 정책
4. 향후 전망

신뢰할 수 있는 출처(뉴스, 보고서)를 기반으로 해 줘.

3-2. 여러 사이트 종합 검색

입력 예시 
"ChatGPT vs Claude vs Gemini" 비교 자료를 찾아 줘.

구글, 네이버, 유튜브에서 각각 검색해서
각 AI의 장단점을 비교한 자료를 종합해 줘.

비교표 형식으로 정리해 줘:
- 가격
- 주요 기능
- 장점
- 단점
- 추천 용도

3-3. 학술 자료 검색

입력 예시 
Google Scholar에서 "machine learning healthcare" 관련
최근 2년간 발표된 논문 10개를 찾아 줘.

각 논문의:
- 제목
- 저자
- 발표 연도
- 핵심 내용 요약
을 정리해 줘.

3-4. 뉴스 모니터링

입력 예시 
오늘 발표된 "삼성전자" 관련 뉴스를 모아 줘.

네이버 뉴스, 구글 뉴스에서 검색해서
주요 뉴스 10개의:
- 제목
- 언론사
- 핵심 내용 (2줄 요약)
을 정리해 줘.

긍정/부정/중립으로 분류도 해 줘.
DAY 4

4일차 · 주식 분석 자동화

🎯 오늘의 목표

마누스 AI로 주식 정보를 수집하고, 분석하고, 보고서를 만드는 방법을 배웁니다.

주식 투자는 본인 책임입니다. 마누스가 제공하는 정보는 참고용이며, 투자 결정은 신중하게 하세요.

4-1. 종목 기본 정보 조회

입력 예시 
네이버 금융에서 "삼성전자" 주식 정보를 조회해 줘.

다음 정보를 정리해 줘:
- 현재가
- 전일 대비 등락률
- 시가총액
- PER, PBR
- 52주 최고/최저
- 외국인/기관 순매수

4-2. 여러 종목 비교 분석

입력 예시 
다음 반도체 관련주 5개를 비교 분석해 줘:
삼성전자, SK하이닉스, DB하이텍, 리노공업, 한미반도체

각 종목별로:
- 현재가
- 시가총액
- PER
- 최근 1개월 수익률
- 애널리스트 목표가

비교표로 정리하고, 투자 매력도 순위도 매겨 줘.

4-3. 뉴스 기반 종목 분석

입력 예시 
"현대자동차" 관련 최근 1주일 뉴스를 분석해 줘.

1. 주요 뉴스 5개 요약
2. 긍정적 뉴스 vs 부정적 뉴스 비율
3. 주가에 미칠 영향 분석
4. 단기 전망 (1주일)

네이버 금융, 한경, 매일경제에서 정보를 수집해 줘.

4-4. 투자 보고서 자동 생성

입력 예시 
"카카오" 종목에 대한 투자 분석 보고서를 만들어 줘.

보고서 구성:
1. 회사 개요
2. 최근 실적 분석
3. 주요 사업부문별 현황
4. 경쟁사 비교
5. SWOT 분석
6. 투자 포인트
7. 리스크 요인
8. 목표가 및 투자의견

A4 2페이지 분량으로 작성해 줘.
DAY 5

5일차 · 블로그 자동 글쓰기

🎯 오늘의 목표

마누스 AI가 블로그 글을 자동으로 작성하고, 실제로 블로그에 포스팅하는 방법을 배웁니다.

5-1. 블로그 글 자동 작성

입력 예시 1 - 정보성 글 
"2024년 최고의 노트북 추천" 주제로 블로그 글을 써 줘.

글 구성:
1. 도입부 (왜 노트북 선택이 중요한지)
2. 용도별 추천 (업무용, 게이밍, 영상편집)
3. 가격대별 추천 (100만원 이하, 100~200만원, 200만원 이상)
4. 구매 시 체크포인트
5. 마무리

SEO를 고려해서 키워드를 자연스럽게 포함해 줘.
2000자 이상으로 작성해 줘.
입력 예시 2 - 리뷰 글 
"갤럭시 S24 울트라" 사용 후기 블로그 글을 써 줘.

포함할 내용:
- 개봉기
- 디자인 및 외관
- 카메라 성능 (사진 예시 설명)
- 배터리 수명
- AI 기능 체험
- 장점과 단점
- 총평 및 추천 대상

실제 사용자 관점에서 생생하게 써 줘.

5-2. 네이버 블로그 포스팅

입력 예시 
내 네이버 블로그에 접속해서
방금 작성한 글을 새 포스트로 등록해 줘.

설정:
- 카테고리: IT/테크
- 공개 설정: 전체 공개
- 태그: 노트북, 노트북추천, 2024노트북

포스팅 완료 후 URL을 알려 줘.
블로그 포스팅 전에 네이버에 로그인되어 있어야 합니다. 마누스가 "권한 요청"을 하면 승인해 주세요.

5-3. 티스토리 블로그 포스팅

입력 예시 
내 티스토리 블로그에 접속해서
다음 내용으로 새 글을 작성해 줘:

제목: [2024] 재택근무 필수 아이템 10선
카테고리: 라이프스타일

글 내용은 재택근무에 도움되는 제품 10가지를
각각 소개하고 추천 이유를 설명해 줘.

HTML 에디터 모드로 작성하고,
적절한 소제목(H2, H3)을 사용해 줘.

5-4. 블로그 시리즈 기획

입력 예시 
"파이썬 기초 강좌" 블로그 시리즈를 기획해 줘.

총 10편 구성으로:
1. 각 편의 제목
2. 다룰 내용 개요
3. 예상 분량
4. 연관 키워드

를 정리하고, 1편 글을 완성해 줘.
DAY 6

6일차 · 쇼핑몰 홈페이지 만들기

🎯 오늘의 목표

마누스 AI가 쇼핑몰 홈페이지를 직접 만들어 주는 과정을 배웁니다.

6-1. 쇼핑몰 기획하기

입력 예시 
핸드메이드 악세서리를 판매하는 쇼핑몰을 기획해 줘.

쇼핑몰 정보:
- 이름: "달빛 공방"
- 컨셉: 감성적이고 따뜻한 느낌
- 주요 상품: 귀걸이, 목걸이, 팔찌
- 타겟: 20~30대 여성

필요한 페이지 구성과 각 페이지에 들어갈 내용을 정리해 줘.

6-2. 홈페이지 제작 요청

입력 예시 
"달빛 공방" 쇼핑몰 웹사이트를 만들어 줘.

페이지 구성:
1. 메인 페이지 (배너, 인기상품, 신상품)
2. 상품 목록 페이지
3. 상품 상세 페이지
4. 장바구니
5. 회사 소개
6. 고객센터

디자인:
- 색상: 파스텔 톤 (연보라, 연핑크)
- 폰트: 부드러운 느낌
- 반응형 (모바일 대응)

HTML, CSS, JavaScript로 만들어 줘.

6-3. 상품 페이지 만들기

입력 예시 
다음 상품들의 상세 페이지를 만들어 줘:

상품 1:
- 이름: 달빛 진주 귀걸이
- 가격: 25,000원
- 설명: 은은한 달빛을 담은 담수진주 귀걸이
- 소재: 14K 금도금, 담수진주
- 사이즈: 길이 3cm

상품 2:
- 이름: 별자리 목걸이
- 가격: 35,000원
- 설명: 나만의 별자리를 담은 목걸이
- 소재: 925 실버
- 사이즈: 체인 45cm

자동 가입 시 개인정보 입력이 필요합니다. 민감한 정보는 직접 입력하시고, 마누스에게는 기본 정보만 맡기세요.
